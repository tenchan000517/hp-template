# 会社概要 ページ詳細設計

## ページ基本情報

**URL**: /about

**meta title**: 会社概要｜〇〇市の外壁塗装・リフォーム【会社名】

**meta description**: 【会社名】の会社概要。〇〇市で創業XX年、地域密着で住まいのお困りごとに対応。代表挨拶、会社情報、沿革をご紹介。

## このページの目的

### 専門家①（マーケティング）の視点

**コンバージョン目標**：
1. 会社への信頼感を醸成し、問い合わせへの心理的障壁を下げる
2. 「地域密着」「歴史」「人」を見せて競合と差別化
3. 会社情報を明示し、実在する信頼できる会社であることを証明

**戦略**：
- 代表の顔と言葉で「人が見える会社」をアピール
- 創業からの歴史で「長く続いている = 信頼できる」
- 地域との関わりを強調し、「地元のパートナー」ポジショニング
- 資格・認定で専門性をアピール

### 専門家②（心理学）の視点

**心理的ゴール**：
1. 「信頼できる会社だ」という確信
2. 「地元に根差した会社」という親近感
3. 「この人たちに任せたい」という人への信頼

**活用する心理メカニズム**：
- 顔の効果：代表・社員の顔を見せることで信頼が増す
- 継続性バイアス：長く続いている = 良いものである
- 近接性バイアス：地域への愛着、地元の会社という安心感
- 権威性：資格・認定・表彰の提示

### 専門家③（デザイン）の視点

**デザイン上の役割**：
- 写真（人物・会社）を効果的に見せ、「人間味」を出す
- 情報量が多いので、整理されたレイアウトが重要
- 温かみのある色調でアプローチ

### セクション2: 代表挨拶

#### なぜこのセクションが必要か

**専門家①（マーケティング）**：
- 代表の顔と言葉で「人が見える会社」をアピール
- 地域への想い、仕事への想いを語ることで共感を生む
- 競合にはない「人間味」で差別化

**専門家②（心理学）**：
- 顔の効果：顔を見せることで信頼が生まれる
- 言葉による共感：価値観の共有で親近感
- 権威性：代表という立場からのメッセージ

- 温かみのある背景色で親しみやすさ
- 引用形式でメッセージを強調

#### レイアウト

**構成**：
ライトベージュ背景に、左に代表者写真、右に挨拶文。

```
┌──────────────────────────────────────────────────────────┐
│ (背景: ライトベージュ #f5f3f0)                           │
│                                                          │
│   「代表挨拶」（H2）                                      │
│                                                          │
│   ┌──────────────┐   ┌──────────────────────────────┐  │
│   │              │   │ 「地域の皆様に選ばれ続けて   │  │
│   │              │   │ XX年。これからも、お客様の   │  │
│   │  [代表写真]  │   │ 住まいと暮らしを守り続けて   │  │
│   │              │   │ まいります。」               │  │
│   │              │   │                              │  │
│   │              │   │ 私たちが大切にしているのは   │  │
│   │              │   │ 「地域密着」です...          │  │
│   │              │   │                              │  │
│   │              │   │ 代表取締役 〇〇 〇〇        │  │
│   └──────────────┘   └──────────────────────────────┘  │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**なぜこのレイアウトか**：
- 左に写真を置くことで、まず「人」を見せる
- 右のテキストは読みやすいカラム幅
- ベージュ背景で温かみと親しみやすさ

**代替案を採用しない理由**：
- 写真なし：人間味が出ない
- 写真小さめ：インパクト不足
- 写真が右：視線が先にテキストに行ってしまう

#### コンテンツ要素

**セクション見出し**：
- テキスト：「代表挨拶」

**代表者写真**：
- 内容：代表者のポートレート写真
- 服装：清潔感のある作業着またはビジネスカジュアル
- 表情：穏やかな笑顔
- サイズ：幅40%（PC）
- アスペクト比：3:4（縦長）
- 角丸：8px

**挨拶文**：
- キャッチ（大きめ）：「地域の皆様に選ばれ続けてXX年。これからも、お客様の住まいと暮らしを守り続けてまいります。」
- 本文（3-4段落）：
  - 段落1：地域密着への想い
  - 段落2：仕事へのこだわり
  - 段落3：お客様への約束
  - 段落4：今後の抱負

**署名**：
- テキスト：「代表取締役 〇〇 〇〇」
- スタイル：手書き風フォント（オプション）または通常フォント

**挨拶文の例**：
```
私たちが大切にしているのは「地域密着」です。
〇〇市で創業してXX年。地元のお客様に支えられ、ここまで歩んでくることができました。

外壁塗装やリフォームは、お客様にとって大きな決断です。だからこそ、私たちは「顔の見える関係」を大切にしています。下請けに任せきりにせず、自社の職人が責任を持って施工する。困ったことがあればすぐに駆けつける。そんな当たり前のことを、愚直に続けています。

お客様の住まいは、人生を過ごす大切な場所です。その住まいを守ることが、私たちの使命だと考えています。

これからも地域の皆様に信頼される会社であり続けるため、社員一同、精進してまいります。
住まいのことでお困りの際は、どうぞお気軽にご相談ください。

代表取締役 〇〇 〇〇
```

#### デザイン詳細

**背景**：
- ライトベージュ（#f5f3f0）
- 理由：温かみ、親しみやすさ、人物写真が映える

**余白**：
- 上：80px
- 下：80px
- 写真とテキストの間：48px

**特徴的な表現**：
- キャッチ部分：24px、ダークネイビー、font-weight 700
- 本文：16px、ダークグレー、line-height 2.0
- 署名：18px、ダークネイビー、右寄せ

#### アニメーション

**使用する**：
- 写真：スクロールで表示時にフェードイン（左から）
- テキスト：0.2秒遅れてフェードイン（右から）

**理由**：
- 写真→テキストの順で視線誘導

#### SP（スマートフォン）での変化

**レイアウト変更**：
- 左右配置 → 縦配置
- 写真を上に、テキストを下に

**サイズ変更**：
- 写真：幅80%、中央揃え
- キャッチ：20px
- 本文：15px

## ページ全体の補足

### 代表者写真の重要性

このページで最も重要な写真は「代表者ポートレート」。以下を満たすこと：

1. **服装**：清潔感のある作業着またはビジネスカジュアル（スーツは堅すぎる）
2. **表情**：穏やかな笑顔（真剣すぎない）
3. **背景**：会社内または工場内（オフィスでも可）
4. **構図**：胸から上、斜め向き、視線はカメラに
5. **ライティング**：自然光または柔らかい照明
