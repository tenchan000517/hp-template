# お問い合わせ ページ詳細設計

## ページ基本情報

**URL**: /contact

**meta title**: お問い合わせ・無料見積もり｜〇〇市の外壁塗装【会社名】

**meta description**: 【会社名】へのお問い合わせ・無料見積もりのご依頼はこちら。お電話またはフォームからお気軽にどうぞ。〇〇市・△△市エリアで外壁塗装・リフォームなら当社へ。

## このページの目的

### 専門家①（マーケティング）の視点

**コンバージョン目標**：
1. 問い合わせフォームの送信完了
2. 電話による問い合わせ
3. LINE等のチャット問い合わせ（オプション）

**戦略**：
- フォームは最小限の項目で離脱を防ぐ
- 電話番号を大きく表示し、電話でも問い合わせしやすく
- 「無料」「強制なし」を明示し、心理的ハードルを下げる
- 送信後の流れを明示し、不安を解消

### 専門家②（心理学）の視点

**心理的ゴール**：
1. 「問い合わせしても大丈夫」という安心感
2. 「簡単に問い合わせできる」という行動容易性
3. 「すぐに返事が来る」という期待感

**活用する心理メカニズム**：
- 損失回避：「無料」「強制なし」で失うものがないことを強調
- 行動容易性：項目数を減らし、入力の手間を軽減
- 期待管理：返答期限を明示し、待つ不安を解消
- 社会的証明：「月間XX件のお問い合わせ」等（オプション）

### 専門家③（デザイン）の視点

**デザイン上の役割**：
- フォームは見やすく、入力しやすく
- 電話番号は目立つように
- 全体的に「安心感」のあるデザイン
- エラー表示は分かりやすく

- 各ページのCTAセクションから
- ヘッダーナビゲーションから
- 「見積もり依頼」ボタンから

**ユーザーの心理状態**：
- 「問い合わせしよう」という決断後
- 「本当に送っていいか」という最後の躊躇
- 「どんな情報を入力するか」という確認

2. 「送っても大丈夫」（安心感）

**理想的なパターン**：
- フォーム送信完了

- 送信完了後のサンクスページへ遷移

## セクション構成

### セクション1: ページヘッダー

#### なぜこのセクションが必要か

**専門家①（マーケティング）**：
- 問い合わせページであることを明示
- 「無料見積もり」を強調

**専門家②（心理学）**：
- 「正しいページにいる」確認
- 「無料」で心理的ハードルを下げる

**専門家③（デザイン）**：
- 他の下層ページと統一感
- ただし背景写真は使わない選択肢も（フォームに集中させるため）

#### レイアウト

**構成**：
高さ200px程度の控えめなヘッダー。シンプルな色背景またはグラデーション。

```
┌──────────────────────────────────────────────────────────┐
│ (背景: メインカラー薄め or グラデーション)               │
│                                                          │
│              「お問い合わせ・無料見積もり」（H1）         │
│           お気軽にご相談ください                         │
│                                                          │
```

**なぜこのレイアウトか**：
- 写真背景ではなくシンプルな色背景で、フォームに集中させる
- 高さ控えめで、すぐにフォームが見える

**代替案を採用しない理由**：
- 写真背景：フォームへの集中が散漫になる
- フルスクリーン：フォームが見えるまで時間がかかる

#### コンテンツ要素

**ページタイトル（H1）**：
- テキスト：「お問い合わせ・無料見積もり」
- サイズ：40px（PC）、28px（SP）

- テキスト：「お気軽にご相談ください」
- サイズ：18px

**代替案を採用しない理由**：
- フォームだけ：補足情報がないと不安
- 補足が上：フォームが下にずれてスクロールが増える

#### コンテンツ要素

**フォーム項目**：

| 項目名 | 入力形式 | 必須/任意 | 備考 |
|-------|---------|----------|------|
| お名前 | テキスト | 必須 | placeholder: 「例）山田 太郎」 |
| ふりがな | テキスト | 任意 | placeholder: 「例）やまだ たろう」 |
| 電話番号 | tel | 必須 | placeholder: 「例）090-1234-5678」 |
| メールアドレス | email | 必須 | placeholder: 「例）example@example.com」 |
| お問い合わせ種別 | ラジオボタン | 必須 | 無料見積もり依頼 / 相談・質問 / その他 |
| ご住所 | テキスト | 任意 | placeholder: 「例）〇〇市〇〇町」見積もり依頼時は入力推奨 |
| お問い合わせ内容 | テキストエリア | 必須 | placeholder: 「ご相談内容をご記入ください」rows: 5 |
| プライバシーポリシー同意 | チェックボックス | 必須 | リンク付き |

**項目数を減らす理由**：
- 項目が多いと離脱率が上がる
- 詳細は電話/メールでヒアリングすればよい
- 最低限の情報で返答可能な設計

**補足情報（右側）**：

```
【このフォームについて】
✓ 無料でご相談いただけます
✓ 強引な営業は一切いたしません
✓ 24時間受付（返答は営業時間内）

【返答目安】
1営業日以内にご連絡いたします。
お急ぎの場合はお電話ください。

📞 0120-XXX-XXX
```

#### デザイン詳細

**背景**：
- オフホワイト（#f8f9fa）

**余白**：
- 上：64px
- 下：64px
- フォームと補足情報の間：48px

**フォーム要素のスタイル**：

- **ラベル**：
  - サイズ：16px、font-weight 600
  - 色：ダークネイビー
  - 必須マーク：赤（*）

- **入力フィールド**：
  - 高さ：48px
  - パディング：12px 16px
  - 枠線：1px solid #e0e0e0
  - 角丸：4px
  - フォーカス時：枠線がメインカラーに、影（box-shadow）

- **テキストエリア**：
  - 高さ：150px（5行程度）
  - リサイズ：縦のみ可能

- **ラジオボタン**：
  - カスタムスタイル（メインカラーの丸）
  - クリック領域を広く

- **チェックボックス**：
  - カスタムスタイル（チェックマーク付き）
  - プライバシーポリシーへのリンク

- **送信ボタン**：
  - 背景：オレンジ（#f39c12）
  - テキスト：白、16px、font-weight 700

  - 送信中：スピナー表示、ボタン無効化

**エラー表示**：
- 入力フィールドの下に赤文字でエラーメッセージ
- エラーのあるフィールドは赤枠線
- 送信ボタンはエラーがあると無効化

**補足情報エリア**：
- 背景：白、角丸8px
- パディング：24px
- チェックマーク：グリーン
- 電話番号：オレンジ

#### アニメーション

**使用する**：
- 送信ボタンクリック時：スピナー表示
- 成功時：サンクスページへ遷移（またはモーダル表示）
- エラー時：エラーフィールドにシェイクアニメーション（軽微）

#### SP（スマートフォン）での変化

**レイアウト変更**：
- 左右配置 → 縦配置

#### コンテンツ要素

**FAQ項目**：

1. **Q. 見積もりは本当に無料ですか？**
   A. はい、お見積もりは完全無料です。現地調査を行い、詳細なお見積書をお渡しします。ご契約を強制することはございませんので、お気軽にご依頼ください。

2. **Q. 対応エリア外でも相談できますか？**
   A. 対応エリア外でも、内容によってはお伺いできる場合がございます。まずはお気軽にご相談ください。

3. **Q. 見積もり後、必ず契約しなければいけませんか？**
   A. いいえ、ご契約を強制することは一切ございません。ご納得いただいた上でご判断ください。

4. **Q. 問い合わせから工事完了までどれくらいかかりますか？**
   A. 工事内容により異なりますが、お問い合わせから現地調査まで1週間以内、お見積り提出から工事着工まで2〜4週間程度が目安です。

5. **Q. 土日祝日でも対応していますか？**
   A. 電話受付は平日9:00-18:00ですが、フォームは24時間受け付けております。土日祝日にいただいたお問い合わせは、翌営業日に対応いたします。

**アコーディオン**：
