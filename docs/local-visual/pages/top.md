# TOP ページ詳細設計

## ページ基本情報

**URL**: /（ルート）

**meta title**: 〇〇市の外壁塗装・リフォームなら【会社名】｜創業XX年・施工実績XXX件

**meta description**: 〇〇市・△△市エリアで外壁塗装・リフォームをお考えなら【会社名】へ。創業XX年、地域密着で施工実績XXX件以上。無料見積もり・現地調査はお気軽にご相談ください。

## このページの目的

### 専門家①（マーケティング）の視点

**コンバージョン目標**：
1. 電話による問い合わせ（最優先）
2. フォームからの見積もり依頼
3. 施工事例ページへの誘導

**達成すべき数値目標**：
- 直帰率 40% 以下
- 電話タップ率（SP） 5% 以上
- 問い合わせフォーム到達率 15% 以上

**戦略**：
- ファーストビューで「地域」「実績」「連絡手段」の3要素を提示し、即座に信頼を獲得
- スクロールするごとに「この会社で間違いない」という確信を強化
- 各セクション末尾にCTAを配置し、どこで決断しても行動できる設計

### 専門家②（心理学）の視点

**心理的ゴール**：
1. 「自分の地域に対応している」という安心感（0-3秒）
2. 「実績がある会社だ」という信頼感（3-10秒）
3. 「今すぐ連絡できる」という行動可能性（10秒以降）

**活用する心理メカニズム**：
- 近接性バイアス：地域名の大きな表示
- 社会的証明：施工実績の数字、施工事例の写真点数
- 権威性：創業年数、資格・認定の表示
- 損失回避：「無料見積もり」「今なら」等のフレーズ

### 専門家③（デザイン）の視点

**デザイン上の役割**：
- サイト全体のトーンを決定する「顔」
- 写真のクオリティと点数でサイト全体の印象が決まる
- 他ページへの期待値を設定する

**視覚的優先順位**：
1. 施工写真（最大面積）
2. 電話番号（最も目立つ色）
3. 実績数字（最大フォントサイズ）
4. 地域名（固定位置で常に視認可能）

## ユーザーの状態

### このページに来る前の状態

**検索クエリ例**：
- 「〇〇市 外壁塗装」
- 「〇〇市 リフォーム おすすめ」
- 「〇〇市 屋根修理 業者」

**ユーザーの心理状態**：
- 「近くで信頼できる業者を探している」
- 「いくつかのサイトを比較している」
- 「すぐに決めたいわけではないが、良さそうなら連絡したい」
- 「価格だけでなく、信頼性も重視している」

**緊急性の高いユーザー**：
- 「雨漏りしている、すぐに来てほしい」
- 「台風で屋根が壊れた」
- → このユーザーには電話番号を即座に見せる必要がある

### このページで感じてほしいこと

1. 「この会社は自分の地域に対応している」（確認）
2. 「実績が多く、信頼できそうだ」（信頼）
3. 「仕事が丁寧そうだ」（品質期待）
4. 「連絡しやすそうだ」（行動意欲）
5. 「他のページも見てみたい」（興味）

### このページから離れる時の状態

**理想的なパターン**：
- 電話をかける
- 問い合わせフォームに移動する
- 施工事例ページに移動する
- 会社概要ページに移動する

**避けたいパターン**：
- 直帰（戻るボタン）
- 競合他社サイトへ移動
- 何も行動せずに離脱

## セクション構成

### セクション1: ファーストビュー（ヒーローエリア）

#### なぜこのセクションが必要か

**専門家①（マーケティング）**：
- ユーザーの50%以上はファーストビューで離脱判断をする
- 3秒以内に「地域」「実績」「連絡手段」を伝えないと競合に流れる
- 電話番号を最初から見せることで、緊急ユーザーを逃さない

**専門家②（心理学）**：
- 第一印象は0.05秒で形成される（プリンストン大学研究）
- 写真の質が「会社の質」の代理指標として判断される
- 大きな数字は「規模」「安定」を連想させる

**専門家③（デザイン）**：
- サイト全体の「格」を決定する最重要セクション
- 写真の選定で「プロフェッショナル」か「素人」かが決まる
- 余白と文字配置で「整理された会社」という印象を作る

#### レイアウト

**構成**：
画面高さ100vh（ビューポート全体）を施工現場の写真が覆う。写真の上に半透明の黒オーバーレイ（50%）を重ね、その上にテキスト要素を配置する。

**写真の要件**：
- 施工完了後の外観写真（晴天、青空背景が理想）
- または職人が作業中の写真（後ろ姿、真剣な表情）
- 解像度：1920x1080px 以上
- 色調：明るく、彩度は自然な範囲

**テキスト配置**：

```
┌──────────────────────────────────────────────────────────┐
│ [地域名: 〇〇市]                          [電話番号]     │
│  32px, white                              36px, orange   │
│                                                          │
│                                                          │
│              [メインキャッチコピー]                       │
│              56px, white, center                         │
│              「地域に根ざして30年。」                     │
│              「お客様の住まいを守り続けます。」           │
│                                                          │
│                                                          │
│    ┌─────────┐    ┌─────────┐    ┌─────────┐           │
│    │ 創業    │    │ 施工    │    │ 地域    │           │
│    │   30年  │    │ 500件+  │    │  密着   │           │
│    │ 72px    │    │ 72px    │    │ 72px    │           │
│    └─────────┘    └─────────┘    └─────────┘           │
│                                                          │
│              [無料見積もりはこちら] ボタン               │
│                                                          │
│                         ↓ (スクロール示唆)               │
└──────────────────────────────────────────────────────────┘
```

**なぜこのレイアウトか**：
- 左上の地域名：視線は左上から始まる（Zパターン）。最初に地域確認を完了させる
- 右上の電話番号：緊急ユーザーは右上を見る習慣がある（ヘッダーCTAの位置）
- 中央のキャッチコピー：視線誘導の終点。「何の会社か」を明確に
- 下部の実績数字：3つ並べることで「充実した実績」を視覚的に表現

**代替案を採用しない理由**：
- 動画背景：読み込み遅延、SP対応の問題、コンテンツ制作コスト
- スライダー：最初の1枚しか見られないデータあり、UXが低下
- 白背景+写真小：インパクト不足、競合と差別化できない

#### コンテンツ要素

**地域名（左上）**：
- テキスト：「〇〇市」または「〇〇市・△△市」
- サイズ：32px（PC）、20px（SP）
- 色：白（#ffffff）
- 配置理由：視線の開始点に配置し、0.5秒以内に地域確認を完了させる

**電話番号（右上）**：
- テキスト：「0120-XXX-XXX」
- サイズ：36px（PC）、24px（SP）
- 色：オレンジ（#f39c12）
- アイコン：電話アイコンを左に配置
- 配置理由：緊急ユーザーが即座に電話できるよう最も目立つ位置に

**メインキャッチコピー（中央）**：
- テキスト例：「地域に根ざして30年。お客様の住まいを守り続けます。」
- サイズ：56px（PC）、28px（SP）
- 色：白（#ffffff）
- font-weight：900
- letter-spacing：0.1em
- 配置理由：視線誘導の終点として、会社のアイデンティティを伝える

**実績数字（中央下部、横並び3つ）**：
- 構成：「創業30年」「施工500件+」「地域密着」
- 数字サイズ：72px（PC）、40px（SP）
- 単位・ラベルサイズ：24px（PC）、16px（SP）
- 色：白（#ffffff）
- 配置：横並び、等間隔（gap: 64px）
- 配置理由：3つの数字で「総合的に信頼できる」印象を形成

**CTAボタン（下部）**：
- テキスト：「無料見積もりはこちら」
- 背景色：オレンジ（#f39c12）
- テキスト色：白（#ffffff）
- サイズ：パディング 16px 48px
- 配置理由：スクロールする前に行動選択肢を提示

**スクロール示唆（最下部）**：
- アイコン：下向き矢印（chevron-down）
- アニメーション：上下に8pxバウンス（1秒ループ）
- 配置理由：「続きがある」ことを示し、スクロールを促す

#### デザイン詳細

**背景**：
- 施工完了写真または作業中写真
- 上に黒オーバーレイ（background: rgba(0,0,0,0.5)）
- 理由：写真を活かしつつ、白文字の可読性を確保

**余白**：
- 上：0px（ヘッダーと接続）
- 下：0px（次セクションと接続）
- 左右：80px（PC）、24px（SP）
- 内部パディング：上下120px（PC）、上下80px（SP）

**特徴的な表現**：
- 写真は object-fit: cover で画面全体にフィット
- 写真の焦点は画面中央やや上（人物の顔や建物の上部が切れないよう）
- テキストは z-index: 10 で写真の上に配置

#### アニメーション

**使用する**：
- 写真：なし（即座に表示）
- テキスト（キャッチコピー、実績数字）：0.3秒後にフェードイン + 上から20pxスライド
- 実績数字：カウントアップアニメーション（0→目標値、1.5秒）
- CTAボタン：0.5秒後にフェードイン

**理由**：
- 写真が主役であることを示すため、写真は動かさない
- テキストを遅延させることで、写真への注目時間を確保
- 数字のカウントアップで「動いている会社」の印象

**具体的な実装**：
```typescript
const fadeSlideIn = {
  initial: { opacity: 0, y: -20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.3, ease: "easeOut", delay: 0.3 }
}
```

#### SP（スマートフォン）での変化

**レイアウト変更**：
- 地域名：左上から中央上に移動
- 電話番号：右上ではなく、ヘッダー固定に統合
- 実績数字：横3列から縦3行に変更
- CTAボタン：幅100%に拡大

**サイズ変更**：
- メインキャッチ：56px → 28px
- 実績数字：72px → 40px
- 内部パディング：120px → 80px

**省略要素**：
- スクロール示唆のアニメーション（SPでは不要）

#### このセクションでの心理効果

**ユーザーが感じること**：
- 「この会社は〇〇市に対応している」（確認完了）
- 「創業30年、500件の実績がある」（数字による信頼）
- 「写真がきれい、仕事も丁寧そう」（視覚的品質評価）

**次のセクションへの期待**：
- 「どんなサービスがあるのか見てみよう」
- 「施工事例をもっと見たい」

### セクション2: サービス概要（3つのポイント）

#### なぜこのセクションが必要か

**専門家①（マーケティング）**：
- ファーストビューで興味を持ったユーザーに「何ができる会社か」を伝える
- 3つに絞ることで記憶に残りやすくする（マジカルナンバー3）
- 詳細ページへの導線を作る

**専門家②（心理学）**：
- 「選択肢が多すぎると決められない」（選択のパラドックス）
- 3つという数は「網羅的だが選びやすい」バランス
- アイコン+テキストで認知負荷を軽減

**専門家③（デザイン）**：
- ファーストビュー（写真大）から情報セクションへの「ギアチェンジ」
- 白背景でリフレッシュし、視線を休ませる
- 均等3分割で安定感を演出

#### レイアウト

**構成**：
白背景に3つのサービスカードを横並び配置。各カードは写真（上）+ アイコン + タイトル + 説明文（下）の構成。

```

│                                                          │
│   ┌────────────┐   ┌────────────┐   ┌────────────┐     │
│   │   [写真]   │   │   [写真]   │   │   [写真]   │     │
│   │            │   │            │   │            │     │
│   │  [アイコン]│   │  [アイコン]│   │  [アイコン]│     │
│   │  外壁塗装  │   │ リフォーム │   │  屋根工事  │     │
│   │  説明文... │   │  説明文... │   │  説明文... │     │
│   │ [詳細→]   │   │ [詳細→]   │   │ [詳細→]   │     │
│   └────────────┘   └────────────┘   └────────────┘     │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**なぜこのレイアウトか**：
- 3カラムは視線が左から右に自然に流れる
- 写真が上にあることで視覚的インパクトを維持
- 均等配置で「どのサービスも同等に重要」を示す

**代替案を採用しない理由**：
- 1カラム（縦積み）：スクロールが長くなりすぎる
- 4カラム以上：1つあたりの面積が小さくなり印象が弱い
- カードなし：情報がまとまらず散漫な印象

#### コンテンツ要素

**セクション見出し**：
- テキスト：「私たちができること」
- サイズ：40px（PC）、24px（SP）
- 色：ダークネイビー（#1a2744）
- 配置：中央揃え

**サービスカード（3つ）**：

各カードの構成：
- 写真：アスペクト比4:3、角丸8px
- アイコン：32x32px、メインカラー
- タイトル：24px（PC）、18px（SP）、font-weight 700
- 説明文：16px（PC）、15px（SP）、2-3行
- 詳細リンク：「詳しく見る →」、オレンジ色

**カード1：外壁塗装**
- 写真：塗装完了後の美しい外壁、または塗装作業中の職人
- アイコン：ペンキ刷毛のアイコン
- タイトル：「外壁塗装」
- 説明文：「美観の回復と建物の保護を両立。お客様のご予算に合わせた塗料選定から施工まで一貫対応。」

**カード2：屋根工事**
- 写真：屋根修理完了後、または職人が屋根で作業中
- アイコン：屋根のアイコン
- タイトル：「屋根工事」
- 説明文：「葺き替え・カバー工法・補修まで対応。雨漏りの根本原因を特定し、確実に解決。」

**カード3：リフォーム**
- 写真：リフォーム完了後の内装、またはビフォーアフター
- アイコン：家のアイコン
- タイトル：「リフォーム」
- 説明文：「キッチン・浴室・トイレなど水回りから、内装・外構まで住まいのことなら何でもご相談。」

#### デザイン詳細

**背景**：
- 白（#ffffff）
- 理由：ファーストビュー（写真ダーク）からの視覚的リフレッシュ

**余白**：
- 上：80px
- 下：80px
- 左右：コンテナ幅（max-width: 1200px）
- カード間：32px
- カード内パディング：24px

**特徴的な表現**：
- カードには影をつけない（フラットデザイン）
- 代わりにホバー時に影が出現（box-shadow: 0 4px 20px rgba(0,0,0,0.1)）
- 写真はカード幅いっぱいに広がる（角丸はカード全体に適用）

#### アニメーション

**使用する**：
- カード：スクロールで画面に入った時、左から順にフェードイン（0.1秒ずつ遅延）
- ホバー時：カード全体が2px上に移動 + 影の出現

**理由**：
- 左から右への視線誘導を強化
- ホバーエフェクトでクリック可能であることを示す

#### SP（スマートフォン）での変化

**レイアウト変更**：
- 3カラム → 1カラム（縦積み）
- カードは縦に3つ並ぶ

**サイズ変更**：
- カード幅：100%
- 写真アスペクト比：4:3 → 16:9（横長に）
- カード間余白：32px → 24px

**省略要素**：
- なし（全要素を表示）

#### このセクションでの心理効果

**ユーザーが感じること**：
- 「外壁塗装以外もやっているんだ」（サービス範囲の理解）
- 「どのサービスも写真がきれい」（品質期待の維持）
- 「自分が必要なサービスがある」（ニーズマッチング）

**次のセクションへの期待**：
- 「実際の施工事例を見たい」
- 「この会社の実績をもっと知りたい」

### セクション3: 施工事例ハイライト

#### なぜこのセクションが必要か

**専門家①（マーケティング）**：
- 「百聞は一見にしかず」—— 施工事例は最強の営業ツール
- トップページで6件見せることで「もっと見たい」欲求を喚起
- 施工事例一覧ページへの導線を作る

**専門家②（心理学）**：
- 社会的証明の可視化（他の人もこの会社を選んでいる）
- ビフォーアフターは「変化」を見せることで効果を実感させる
- 地域名付きの事例は「同じ地域の人も利用」という安心感

**専門家③（デザイン）**：
- 写真グリッドによる「量感」の演出
- トップページ全体の中で最も写真密度が高いセクション
- 視覚的なクライマックス（ピーク）を作る

#### レイアウト

**構成**：
オフホワイト背景に、セクション見出し + 6件の施工事例グリッド + 「もっと見る」ボタン。

| 深度 | セクション | ユーザーの心理状態 | 設計意図 |

セクション1: 写真ダーク（オーバーレイ）
    ↓

    ↓
セクション3: オフホワイト
