# サービス ページ詳細設計

## ページ基本情報

**URL**: /service

**meta title**: 外壁塗装・屋根工事・リフォームサービス｜〇〇市の【会社名】

**meta description**: 【会社名】のサービス一覧。外壁塗装・屋根工事・リフォームなど、住まいのお困りごとに幅広く対応。〇〇市を中心に地域密着で施工。無料見積もりはお気軽にどうぞ。

## このページの目的

### 専門家①（マーケティング）の視点

**コンバージョン目標**：
1. 各サービスの詳細ページへの誘導
2. 問い合わせフォームへの誘導
3. 「このサービスがほしい」という明確なニーズ形成

**戦略**：
- サービスごとに写真を大きく見せ、「仕上がりの質」を視覚的に伝える
- 各サービスに特徴・流れ・価格目安を含め、比較検討に必要な情報を提供
- ページ下部で「どれを選べばいいかわからない」ユーザーへの相談誘導

### 専門家②（心理学）の視点

**心理的ゴール**：
1. 「自分が必要なサービスがある」という確認
2. 「このクオリティなら頼みたい」という品質期待
3. 「料金が明確で安心」という価格への不安解消

**活用する心理メカニズム**：
- カテゴライゼーション：サービスを明確に分類し、選択を容易に
- アンカリング：価格帯の提示で予算感を形成
- 権威性：資格・認定の表示でプロフェッショナル感

### 専門家③（デザイン）の視点

**デザイン上の役割**：
- 各サービスを視覚的に差別化（写真のトーン、アイコン）
- 情報量が多いページなので、整理されたレイアウトが重要
- 写真の質でサービスの質を代弁する

## セクション構成

### セクション1: ページヘッダー（サービス一覧導入）

#### なぜこのセクションが必要か

**専門家①（マーケティング）**：
- ページの内容を即座に伝える
- 「このページで何がわかるか」を明確に
- サービス選択への心理的準備

**専門家②（心理学）**：
- ページ遷移後の「ここで合っているか」確認
- 期待値の設定

**専門家③（デザイン）**：
- ページのトーンを設定
- 視覚的な区切りを作る
- TOPページのファーストビューとは異なる「落ち着いた」印象

#### レイアウト

**構成**：
高さ300px程度の帯状セクション。背景には施工現場の写真（ぼかし処理）を配置し、中央にページタイトルと説明文。

```
┌──────────────────────────────────────────────────────────┐
│ (背景: 施工写真 + ぼかし + 暗めオーバーレイ)             │
│                                                          │
│              「サービス一覧」（H1）                       │
│              住まいのお困りごと、何でもご相談ください     │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**なぜこのレイアウトか**：
- 下層ページでも写真を使うことでサイト全体のトーンを維持
- 高さを控えめにすることで「ここは導入部」と示す
- ぼかし処理で写真がテキストを邪魔しない

**代替案を採用しない理由**：
- 白背景のシンプルヘッダー：TOPとの差が大きすぎる
- フルスクリーンヒーロー：サービス一覧が見えるまで時間がかかる

#### コンテンツ要素

**ページタイトル（H1）**：
- テキスト：「サービス一覧」
- サイズ：48px（PC）、32px（SP）
- 色：白（#ffffff）
- font-weight：700

**サブタイトル**：
- テキスト：「住まいのお困りごと、何でもご相談ください」
- サイズ：18px（PC）、16px（SP）
- 色：白の80%透明度

#### デザイン詳細

**背景**：
- 施工中の写真（職人の作業風景など）
- filter: blur(4px) でぼかし処理
- 黒オーバーレイ 60%
- 理由：テキストの可読性確保、TOPページより落ち着いた印象

**余白**：
- 上：120px（ヘッダー高さを含む）
- 下：60px
- 左右：コンテナ幅

**特徴的な表現**：
- 写真は固定背景（background-attachment: fixed）でパララックス風
- ただしSPでは無効化（パフォーマンス考慮）

#### アニメーション

**使用しない**：
- 理由：導入部は静かに、コンテンツに注目させる

#### SP（スマートフォン）での変化

**レイアウト変更**：
- 高さ：300px → 200px
- 背景固定は無効化

**サイズ変更**：
- タイトル：48px → 32px

#### このセクションでの心理効果

**ユーザーが感じること**：
- 「サービス一覧ページに来た」（確認）
- 「落ち着いたトーンで信頼できそう」

**次のセクションへの期待**：
- 「どんなサービスがあるか見よう」

**専門家②（心理学）**：

- 「安さ」よりも「明瞭さ」を重視
