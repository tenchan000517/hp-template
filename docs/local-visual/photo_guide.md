# Local Visual 撮影ディレクション

このテンプレートで使用する全ての画像素材を網羅的にリストアップし、撮影指示を記載する。

---

## 1. 撮影トーン＆マナー

### 全体の方向性

| 項目 | 内容 |
|-----|------|
| キーワード | 信頼感、地域密着、プロフェッショナル、明るい、清潔、自然体 |
| 目指すイメージ | 「この会社なら安心して任せられる」「地元で長く続いている」 |
| ライティング | 自然光を活かした明るさ、または自然光に近い照明 |
| 色味 | 暖かすぎず寒すぎない、清潔感のあるニュートラルなトーン |
| 全体のトーン | 爽やかで誠実、技術力を感じさせつつも親しみやすい |

### OK / NG

| OK | NG |
|----|-----|
| 晴天時の屋外撮影 | 曇天・雨天時の暗い写真 |
| 自然光を活かした明るさ | 暗い照明、強すぎる影 |
| 施工後のきれいな仕上がり | 施工途中の汚れた状態 |
| 整理整頓された現場 | 雑然とした現場 |
| 作業中の真剣な表情 | ポーズを取った不自然な表情 |
| 作業着姿の職人 | スーツ姿（代表者以外） |
| 清潔な作業着 | 汚れすぎた作業着 |
| 安全装備着用の様子 | 安全装備なしの危険な作業 |

### 撮影時の注意事項

1. **天候**: 晴天時を選ぶ。青空が映る外観写真が理想
2. **時間帯**: 10:00-15:00 が理想。朝夕は影が長くなりすぎる
3. **安全**: ヘルメット、安全靴着用を徹底
4. **整理整頓**: 撮影前に現場を片付ける
5. **服装**: 作業着は清潔なものを用意
6. **表情**: 真剣な表情と笑顔の両パターンを撮影

| シーン | アングル | 構図 | ポイント |

## 5. 施工事例撮影指示

### Before/After 撮影ルール

| 項目 | Before | After |
|-----|--------|-------|
| アングル | **完全に同一** | **完全に同一** |
| 時間帯 | 午前または午後（統一） | **同じ時間帯** |
| 天候 | 晴天推奨 | **同じ天候** |
| カメラ位置 | 三脚使用推奨、位置をマーキング | **同じ位置** |
| 構図 | 建物全体が入る | **同じ構図** |

### Before撮影のポイント

- 劣化状態がわかるように（チョーキング、色褪せ、ひび割れ）
- ただし「ひどすぎる」印象にならないバランス
- 汚れや雑草は撮影前に最低限整理

### After撮影のポイント

- 施工直後の美しい状態で撮影
- 足場解体後、清掃完了後に撮影
- 青空背景が理想
- 塗料の艶感が伝わるように

### ギャラリー写真（1事例あたり3-5枚）

1. 高圧洗浄の様子
2. 下地処理/塗装作業中
3. 細部の仕上げ
4. 施工完了後のアップ（壁面の質感）
5. 施工完了後の別アングル

## 8. AI画像生成の場合

撮影が難しい場合、AI画像生成（Nanobanana等）を使用する。

### 生成時の注意

1. **日本の住宅**: 日本の戸建て住宅スタイル（洋風ではなく和モダン等）
2. **日本人職人**: アジア人の職人、日本の作業着スタイル
3. **天候**: 晴天、青空
4. **品質**: 高解像度（4K以上）
5. **一貫性**: 同じ建物のBefore/Afterは同一モデルで生成
6. **ロゴ・文字**: 画像内にロゴや文字を入れない（後から合成）

### プロンプト例

**施工完了後の外観**:
```
Japanese modern house exterior, freshly painted walls, beige color, clear blue sky,
sunny day, residential area, high quality architectural photography,
realistic, 4k, sharp focus
```

**職人ポートレート**:
```
Japanese male construction worker in his 40s, wearing clean navy work clothes,
confident smile, arms crossed, construction site background,
portrait photography, natural lighting, professional
```

**塗装作業中**:
```
Japanese painter working on house exterior wall, using paint roller,
back view, blue sky, scaffolding visible, focused work,
documentary style photography, natural lighting
```
