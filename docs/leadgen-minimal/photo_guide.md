# LeadGen Minimal 撮影ディレクション

## 概要

LeadGen Minimal テンプレートは **テキスト中心** の設計であり、必要な画像素材は最小限に抑えられている。
ただし、以下の素材は必須または推奨される。

このドキュメントでは、サイト完成に必要な全ての素材をリストアップする。

---

## 5-1. 撮影トーン＆マナー

### 基本方針

**コンセプト**: 「静謐な自信」

派手な演出や過剰な装飾を避け、**品質と信頼を視覚で伝える**。

### トーンの特徴

| 項目 | 方向性 |
|------|--------|
| 色調 | ナチュラル、わずかにクールに。彩度は抑えめ |
| 明るさ | やや明るめ。ハイキーすぎない |
| コントラスト | 控えめ。柔らかい光を使用 |
| 雰囲気 | 落ち着いた、プロフェッショナルな印象 |
| 避けるもの | ビビッドな色、過度な加工、派手なポーズ |

### 撮影スタイル

**人物撮影**:
- 自然光またはソフトボックスで柔らかく
- 背景はシンプル（白、グレー、オフィス環境）
- 表情は穏やかで、信頼感のある微笑み
- ビジネスカジュアルまたはスーツ

**オフィス撮影**:
- 整理整頓された環境
- 自然光を活かした撮影
- 余計な物が映り込まないよう注意

## 5-3. 撮影アングル・構図指示

### ロゴ撮影（デザイン指示）

**形式**: SVGベクター形式必須

**バリエーション**:
1. **フルロゴ**: 会社名 + シンボル
2. **シンボルのみ**: ファビコン用
3. **横長バージョン**: ヘッダー用（高さ40px程度で収まるように）

**カラーバリエーション**:
1. **ダークネイビー版**: 白背景用（#1a1a2e）
2. **白版**: ダーク背景用（#FFFFFF）
3. **モノクロ版**: 印刷物用

### OGP画像撮影（デザイン指示）

**サイズ**: 1200 x 630 px

**構成**:
- 背景: ダークネイビー（#1a1a2e）またはオフホワイト（#f8f8f8）
- 中央: 会社ロゴ
- ロゴ下: キャッチコピー（任意）
- 余白: 十分に確保（周囲10%以上）

**禁止事項**:
- 過度な装飾
- グラデーション
- 複数の画像の合成

### 代表者写真（任意）

**アングル**: 正面からやや斜め45度

**フレーミング**: バストアップ（胸から上）

**表情**: 穏やかな微笑み。歯は見せても見せなくてもOK

**服装**: スーツまたはビジネスカジュアル。ネクタイは任意

**背景**:
- 白またはライトグレーの無地
- オフィス内（ぼかし入り）

**ライティング**: ソフトボックスまたは窓際の自然光

## 5-5. 製品撮影指示

**注意**: LeadGen Minimal テンプレートでは物理的な製品撮影は想定していない。

サービス業向けのテンプレートであり、製品ではなく「人」と「成果」で訴求する設計。

製品撮影が必要な場合は、別途指示書を作成すること。

## AI画像生成時の注意

実際の撮影ができない場合、AI画像生成で代替することも可能。
ただし、以下の点に注意：

1. **人物のAI生成は避ける**: 特に代表者写真は実写を推奨。AI生成の人物は信頼性を損なう可能性がある
2. **抽象的なイメージに限定**: 幾何学模様、テクスチャ、風景などはAI生成でもOK
3. **生成後の調整**: 色調をサイトのカラーパレットに合わせる
4. **著作権確認**: 商用利用可能なサービスを使用すること

---

## 補足: このテンプレートで画像を使わない理由

LeadGen Minimal は意図的に画像を最小限にしている。理由は以下：

1. **高速な読み込み**: 画像が少ないほどページ表示が速い
2. **情報の集中**: テキストに視線を集中させる
3. **Minimal表現**: 「削ぎ落とす」コンセプトの体現
4. **運用のしやすさ**: 画像更新の手間を削減
5. **汎用性**: どの業種にも適用しやすい

画像を追加する場合は、本当に必要かを検討すること。
「画像があった方が良い」ではなく「画像がないと成り立たない」場合のみ追加する。
