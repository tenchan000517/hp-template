# 撮影ディレクション

## 1. 撮影トーン＆マナー

### 全体の方向性

| 項目 | 内容 |
|-----|------|
| キーワード | 信頼、実績、プロフェッショナル、誠実、活気 |
| 目指すイメージ | 「この会社に任せたい」「安心して取引できる」と思わせる |
| ライティング | 自然光を活かした明るさ、または自然光に近い照明 |
| 色味 | 暖かすぎず、寒すぎず。清潔感のあるニュートラルなトーン |
| 全体的な印象 | 「きちんとした会社」「活動している会社」「人が見える会社」 |

### OK / NG

| OK | NG |
|----|-----|
| 自然光を活かした明るい写真 | 暗い照明、影が強すぎる写真 |
| 作業中の真剣な表情 | ポーズを取った不自然な表情 |
| 会話中の自然な笑顔 | 作られた笑顔、過剰な演出 |
| 整理整頓された清潔な背景 | 雑然とした背景、不要な物が映り込む |
| 動きのある瞬間 | 静止した硬いポーズ |
| 実際の作業・成果物 | ストックフォト感のある写真 |
| 顔がはっきり見える構図 | 顔が暗い、シルエットになっている |

### Trust Visual 特有の注意点

1. **顔写真は必ず「顔が認識できる」サイズで撮影**: お客様の声、代表者写真は特に重要
2. **数字の裏付けとなる写真を意識**: 「12,000件の実績」なら、実績物の多様さを感じさせる
3. **「実在感」を重視**: ストックフォト風にならないよう、リアルな現場感を出す
4. **認証バッジ・資格証は原物を撮影**: ロゴデータがあればそちらを使用

## 3. 撮影アングル・構図指示

### 人物撮影

| シーン | アングル | 構図 | ポイント |
|-------|---------|------|---------|
| 代表者ポートレート | 目線の高さ | 被写体を中央やや右に配置 | 背景はボケさせる（F2.8程度）。威厳と親しみやすさのバランス。 |
| お客様ポートレート | 目線の高さ | 被写体を1/3ルールで配置 | 胸から上。顔がはっきり見える明るさ。自然な表情。 |
| 作業中 | やや低め（仰ぎ見る角度） | 手元と顔が両方入る | 真剣な表情を捉える。動きのある瞬間。 |
| 打ち合わせ | 目線の高さ | 複数人が自然に収まる | 会話の瞬間を狙う。硬くならないように。 |

### 設備・製品撮影

| シーン | アングル | 構図 | ポイント |
|-------|---------|------|---------|
| 設備全景 | やや低め | 対角線構図 | 迫力と規模感。人を入れてスケール感を出す。 |
| 作業中 | 水平または斜め | 中央に被写体 | ディテールと動き。稼働中の臨場感。 |
| 製品クローズアップ | 真上 or 45度 | 中央配置 | 影を調整、質感重視。品質の高さが伝わるように。 |
| 納品物（実績） | 斜め45度 | 全体が見える構図 | 明るく鮮明に。背景はシンプルに。 |

### 会社外観

| アングル | 構図 | ポイント |
|---------|------|---------|
| 正面または斜め前 | 建物全体が入る | 天気の良い日。看板・社名が読める。電線など邪魔なものを避ける。 |

## 8. AI画像生成用の指示（Nanobanana等）

### 生成時のポイント

1. **人物写真はAI生成を避ける**: お客様の声、代表者写真は必ず実写を使用
2. **製品・実績写真もできる限り実写**: AI生成は最終手段
3. **背景・イメージ写真のみAI生成を検討**: 抽象的な背景、パターン等

### AI生成可能な素材

- 幾何学パターン（ページヒーロー背景用）
- テクスチャ背景
- 抽象的なビジネスイメージ（ただし使用は最小限に）

### AI生成不可の素材

- お客様の顔写真（実在しない人物は信頼を損なう）
- 代表者・社員の顔写真
- 実績・納品物の写真（嘘になる）
- 会社外観（実在しない建物は問題）

## 9. 撮影チェックリスト

- [ ] 撮影許可の取得（社内、お客様）

- [ ] 全カットのバックアップ
