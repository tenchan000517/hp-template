# 会社概要ページ 詳細設計

## 基本情報

**URL**: `/about`

**ファイル**: `src/app/about/page.tsx`

## このページの目的

### 👤 専門家①（マーケティング）視点

**マーケティング目標**:
1. 会社の基本情報を正確に伝え、「実在する会社」であることを証明する
2. 資格・認証・所属団体等で「公的に認められた会社」であることを示す
3. 代表者の顔と想いを見せ、「人が見える会社」として信頼を構築する

**KPI指標**:
- ページ完読率: 50%以上
- お問い合わせへの遷移率: 8%以上
- 代表メッセージまでのスクロール率: 60%以上

### 👤 専門家②（心理学）視点

**心理的ゴール**:
1. 「所在地」「設立年」「資本金」等の基本情報で「実在性」を確認させる
2. ISO認証、業界団体等の「お墨付き」で権威性を付与する
3. 代表者の顔写真とメッセージで「誠実さ」を伝える

**活用する心理メカニズム**:
- **権威性バイアス**: 資格、認証、受賞歴は信頼の源泉
- **顔認識効果**: 代表者の顔を見ることで親近感と信頼が生まれる
- **具体性効果**: 住所、電話番号、設立年等の具体情報は信頼を高める

### 👤 専門家③（デザイン）視点

**デザイン上の役割**:
1. 情報を整然と整理し、「きちんとした会社」という印象を与える
2. 資格バッジを目立つ位置に配置し、「お墨付き」を視覚化
3. 代表者写真を大きく表示し、「人」を前面に出す

## ユーザーの状態

**このページに来る前の状態**:
- 他のページで信頼を感じ、最終確認として会社情報を見たい
- 「本当に実在する会社か」「どこにあるのか」を確認したい
- 問い合わせ前の最終チェックとして訪問

**このページで感じてほしいこと**:
- 「しっかりした会社だな」
- 「ISO認証も取っているなら安心だ」
- 「代表の想いがわかると、信頼できる」

**このページから離れる時の状態**:
- お問い合わせへ遷移: 「安心した。問い合わせよう」
- サービスページへ遷移: 「サービス内容を再確認したい」

## セクション構成

会社概要ページは以下の5セクションで構成する:

1. ページヒーロー（Page Hero）: ページタイトル + 会社外観写真
2. 会社情報テーブル（Company Info）: 基本情報の一覧
3. 資格・認証（Certifications）: 取得資格・認証バッジ
4. 代表メッセージ（CEO Message）: 代表者の写真とメッセージ
5. アクセス（Access）: 地図と交通案内

## セクション1: ページヒーロー（Page Hero）

### なぜこのセクションが必要か

**👤 専門家①（マーケティング）**:
会社の外観写真を見せることで「実在する会社」であることを視覚的に証明する。建物が見えると「ここに行けば会える」という安心感が生まれる。

**👤 専門家②（心理学）**:
人間は「見たことがあるもの」に対して親近感を持つ（単純接触効果）。外観写真を見せることで、初対面でも「知っている感」を与える。

**👤 専門家③（デザイン）**:
外観写真をフルワイドで配置し、インパクトを与える。写真の上にタイトルをオーバーレイし、一目で「会社概要ページ」とわかるようにする。

### レイアウト

**構成**:
- 高さ: 50vh
- 背景: 会社外観写真 + 黒オーバーレイ（opacity 0.4）
- ページタイトル: 白文字、中央配置、56px
- サブタイトル: 「COMPANY」（英語表記、16px、白）

**なぜこのレイアウトか**:
- 外観写真で「実在する会社」を視覚的に証明
- タイトルをオーバーレイすることで、写真とタイトルを同時に見せる
- 50vhで本題（会社情報）に早く入る

### コンテンツ要素

**背景写真**:
- 内容: 会社の外観写真（建物全体、または入口）
- 要件: 明るく清潔な印象、天気の良い日に撮影
- サイズ: 1920×1080px以上

**ページタイトル**:
- テキスト: 「会社概要」
- サイズ: 56px（PC）、40px（SP）
- 色: 白

**サブタイトル**:
- テキスト: 「COMPANY」
- サイズ: 16px
- 色: 白（opacity 0.8）
- 配置: タイトルの16px上

### デザイン詳細

**背景**:
- 構成: 外観写真 + 黒オーバーレイ（opacity 0.4）

**余白**:
- 上余白: 80px
- 下余白: 60px

### SP（スマートフォン）での変化

**サイズ変更**:
- タイトル: 40px
- 高さ: 40vh

## セクション2: 会社情報テーブル（Company Info）

### なぜこのセクションが必要か

**👤 専門家①（マーケティング）**:
会社の基本情報を正確に伝える。特に「設立年」「資本金」「従業員数」は会社の規模と安定性を示す重要な指標。

**👤 専門家②（心理学）**:
具体的な情報（住所、電話番号、設立年）は「隠し事がない」という印象を与える。情報が多いほど「透明性が高い」と感じられる。

**👤 専門家③（デザイン）**:
テーブル形式で情報を整理し、読みやすくする。左にラベル、右に値という一貫したフォーマットで、視認性を高める。

### レイアウト

**構成**:
- 背景色: 白（#FFFFFF）
- セクション見出し: 「基本情報」（左揃え、32px）
- 情報テーブル: ラベル（左）+ 値（右）の2カラム

**テーブルの配置**:
- 左右マージン: 各15%（コンテンツ幅70%）
- ラベル列: 25%幅
- 値列: 75%幅
- 各行のパディング: 上下20px
- 行間の区切り: 1px線（#e8e8e8）

**なぜこのレイアウトか**:
- テーブル形式で情報を整理し、スキャンしやすくする
- ラベルを狭く、値を広くすることで、内容を読みやすく
- 区切り線で行を区別し、読み間違いを防ぐ

## セクション3: 資格・認証（Certifications）

### なぜこのセクションが必要か

**👤 専門家①（マーケティング）**:
ISO認証、業界資格、受賞歴等を見せることで「第三者に認められた会社」であることを証明する。自社のアピールより、公的な認証の方が説得力がある。

**👤 専門家②（心理学）**:
権威性バイアスを最大限に活用する。「ISO9001」「〇〇協会認定」等のバッジは、「専門家が認めた品質」の証拠として機能する。

**👤 専門家③（デザイン）**:
認証バッジを大きく、目立つ位置に配置する。バッジは原色のまま使用し、権威性を損なわない。背景はオフホワイトにしてバッジを際立たせる。

### レイアウト

**構成**:
- 背景色: オフホワイト（#f8f8f8）
- セクション見出し: 「資格・認証」（中央揃え、32px）
- 認証バッジ: 4-6個を横並び（PC）

**バッジの配置**:
- 中央揃え
- 各バッジの高さ: 80px
- バッジ間のギャップ: 48px
- バッジの下に認証名を表示

**なぜこのレイアウトか**:
- バッジを大きく表示することで「お墨付き」感を強調
- 横並びにすることで「複数の認証がある」ことを一目で伝える
- 原色のまま使用することで、権威性を損なわない

### コンテンツ要素

**セクション見出し**:
- テキスト: 「資格・認証」
- サイズ: 32px（PC）、24px（SP）
- 色: メインカラー
- 配置: 中央揃え

**認証バッジ（例）**:

| 認証名 | バッジ画像 | 取得年 |

| ISO9001 | iso9001.png | 2010年取得 |

| 〇〇協会認定 | association.png | 2018年認定 |
| △△技能士（5名在籍） | skill.png | - |

**各バッジの構成**:
- バッジ画像: 80px高さ
- 認証名: 14px、メインカラー、Bold
- 補足: 12px、サブテキストカラー（取得年等）

### デザイン詳細

**背景**:
- 色: オフホワイト（#f8f8f8）

**余白**:
- 上余白: 80px
- 下余白: 80px

## セクション4: 代表メッセージ（CEO Message）

### なぜこのセクションが必要か

**👤 専門家①（マーケティング）**:
代表者の顔と言葉を見せることで「人が見える会社」として信頼を構築する。特にBtoBでは「誰と取引するか」が重要な判断基準になる。

**👤 専門家②（心理学）**:
人間は顔を認識することで親近感と信頼を感じる（顔認識効果）。代表者の顔写真とメッセージを見せることで、「この人なら信頼できそう」という直感的な判断を促す。

**👤 専門家③（デザイン）**:
代表者の写真を大きく配置し、「人」を前面に出す。メッセージは読みやすい行間で、誠実さを感じさせるトーンで表現する。

### レイアウト

**構成**:
- 背景色: 白（#FFFFFF）
- 左側40%: 代表者写真（縦長、ビジネスポートレート）
- 右側60%: 「代表メッセージ」見出し + メッセージ本文 + 署名

**なぜこのレイアウトか**:
- 写真を大きく見せることで「顔が見える」安心感を与える
- 左に写真、右にテキストという構成で、視線を自然に誘導
- 署名（氏名）を添えることで、責任の所在を明確にする

### コンテンツ要素

**代表者写真**:
- 内容: ビジネスポートレート（スーツまたは作業着）
