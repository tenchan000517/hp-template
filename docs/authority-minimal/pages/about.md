# 会社概要ページ 詳細設計

**URL**: /about

---

## このページの目的

### 専門家①（マーケティング）の視点

**マーケティング目標:**
- 会社の基本情報を過不足なく提供
- 信頼性の証明（法人としての実在）
- 歴史・継続性による権威づけ
- 所在地の明示による安心感

**コンバージョン設計:**
- 直接的なCVは期待しない
- 「ちゃんとした会社」という確認のページ
- 問い合わせ前の最終確認として機能

### 専門家②（心理学）の視点

**心理的ゴール:**
- **実在の確認**: 法人として存在するという安心感
- **歴史による権威**: 「〇年続いている」という信頼
- **透明性**: 情報を隠さない姿勢への好感

**活用する心理効果:**
1. 実体の確認: 住所・連絡先があることで安心
2. 継続性の権威: 創業年数は信頼の証
3. 情報開示: 隠さないことで信頼を得る

### 専門家③（デザイン）の視点

**デザイン上の役割:**
- 事務的な情報を、品格を損なわず提示
- 表組みは使うが、Excelのような見た目にしない
- 余白を活かし、Authority Minimalのトーンを維持

**視覚的ゴール:**
- 情報は整然と、しかし堅すぎず
- 必要最小限の情報のみ
- オフィス写真があれば1枚だけ添える

---

## ユーザーの状態

**このページに来る前の状態:**
- サービス内容、事例を理解した
- 問い合わせの前に「会社の基本情報」を確認したい
- または、取引先審査のために情報を収集中

**このページで感じてほしいこと:**
- 「ちゃんとした会社だ」
- 「必要な情報がすべてある」
- 「信頼できる」

**このページから離れる時の状態:**
- 問い合わせページへ: 「確認できたので相談しよう」
- 離脱: 情報収集が完了

---

## セクション構成

会社概要ページは以下の4セクションで構成する:

1. **ページタイトル**: シンプルな見出し
2. **会社情報**: 基本情報一覧
3. **沿革**: 会社の歴史（任意）
4. **アクセス**: 所在地と地図（任意）

---

## セクション2: 会社情報

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
取引先の確認、与信審査、採用検討など、様々な目的で会社情報が求められる。必要十分な情報を、見つけやすく提示する。

**専門家②（心理学）:**
会社情報の透明性は信頼に直結。特に「代表者名」「住所」「設立年」は重要。資本金は業種によって記載の要否を判断。

**専門家③（デザイン）:**
一般的なdl/dt/dd形式だが、余白とタイポグラフィで品格を出す。表の罫線は使わず、余白のみで区切る。

### レイアウト

**構成:**
ラベルと値を左右に配置するシンプルな形式。

```
+--------------------------------------------------+
|                                                  |
|   Company                                        |
|   会社情報                                        |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   会社名         株式会社〇〇                     |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   代表者         山田 太郎                        |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   設立           2003年4月                        |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   所在地         〒000-0000                       |
|                  東京都渋谷区〇〇1-2-3            |
|                  〇〇ビル 5F                      |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   事業内容       ブランディング                   |
|                  ウェブサイト制作                 |
|                  コンサルティング                 |
|                                                  |
+--------------------------------------------------+
```

**なぜこのレイアウトか:**
- ラベルと値を左右に並べ、一覧性を確保
- 区切り線で各項目を分離
- 複数行の値（住所、事業内容）も自然に表示

### コンテンツ要素

**セクションタイトル**
- 英語: 14px、Cormorant Garamond、#999999
- 日本語: 24px、游明朝、#1A1A1A

**情報項目**

ラベル:
- サイズ: 14px、游ゴシック、#999999
- 幅: 120px固定
- 位置: 左

値:
- サイズ: 15px、游ゴシック、#333333
- 位置: ラベルの右側
- 行間: 1.8（複数行の場合）

区切り線:
- 幅: 100%、高さ: 1px、色: #E0E0E0
- 各項目の上に配置

**掲載する項目（推奨）:**

| 項目 | 必須/任意 | 備考 |
|------|----------|------|
| 会社名 | 必須 | 正式名称（株式会社を含む） |
| 代表者 | 必須 | 代表取締役名 |
| 設立 | 必須 | 設立年月 |
| 所在地 | 必須 | 郵便番号、住所、ビル名 |
| 電話番号 | 任意 | 掲載しない選択肢もあり |
| 事業内容 | 必須 | 3-5項目 |
| 資本金 | 任意 | 業種による |
| 従業員数 | 任意 | 少数精鋭の場合は省略可 |
| 取引銀行 | 任意 | 与信目的で必要な場合 |

### デザイン詳細

**背景:** #FFFFFF

**余白:**
- 上余白: 100px
- 各項目の上下パディング: 24px
- 左右マージン: 15%

**特徴的な表現:**
- 表の罫線ではなく、区切り線
- ラベルはグレーで控えめに
- 値は黒で明確に

### アニメーション

**使用する/しない:** 使用しない

### SP（スマートフォン）での変化

**レイアウト変更:**
- ラベルと値を縦に積む（ラベル上、値下）

**サイズ変更:**
- ラベル: 13px
- 値: 14px
- 左右パディング: 24px

## セクション3: 沿革（任意）

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
創業からの歴史は信頼の証。特に長い歴史がある場合、重要な権威づけになる。ただし、創業数年の場合は省略可。

**専門家②（心理学）:**
時間軸での成長は「継続性」「安定性」を示す。ただし、羅列が長すぎると読まれない。5-10項目に絞る。

**専門家③（デザイン）:**
タイムライン形式だが、過度な装飾は避ける。年号と出来事をシンプルに並べる。

### レイアウト

**構成:**

```
+--------------------------------------------------+
|                                                  |
|   History                                        |
|   沿革                                            |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   2003        創業。代表 山田が個人事業として開始 |
|                                                  |
|   2005        株式会社〇〇 設立                   |
|                                                  |
|   2010        渋谷区に事務所移転                  |
|                                                  |
|   2015        □□賞 受賞                         |
|                                                  |
|   2020        従業員10名体制に                    |
|                                                  |
|   2024        設立20周年                          |
|                                                  |
+--------------------------------------------------+
```

### コンテンツ要素

**年号:**
- サイズ: 14px、Cormorant Garamond、#999999
- 幅: 80px固定

**出来事:**
- サイズ: 15px、游ゴシック、#333333
- 年号の右に配置

**項目間余白:** 24px

### デザイン詳細

**背景:** #F5F5F5
**余白:** 上下各120px

### SP（スマートフォン）での変化

**レイアウト変更:**
- 年号と出来事を縦に積む
- 左パディング: 24px

## セクション4: アクセス（任意）

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
来社を想定する業種では必須。地図があることで「実在する会社」という安心感を与える。

**専門家②（心理学）:**
所在地の可視化は信頼性の証明。ただし、リモートワーク主体の場合は省略も可。

**専門家③（デザイン）:**
Google Mapsの埋め込みは品位を損なわないよう、余白で囲む。または、画像としての地図を使用。

### レイアウト

**構成:**

```
+--------------------------------------------------+
|                                                  |
|   Access                                         |
|   アクセス                                        |
|                                                  |
|   +--------------------------------------------+ |
|   |                                            | |
|   |           [Google Maps埋め込み]             | |
|   |           または 地図画像                   | |
|   |                                            | |
|   +--------------------------------------------+ |
|                                                  |
|   〒000-0000                                      |
|   東京都渋谷区〇〇1-2-3 〇〇ビル 5F               |
|                                                  |
|   〇〇駅 徒歩5分                                  |
|                                                  |
+--------------------------------------------------+
```

### コンテンツ要素

**地図:**
- 幅: 100%
- 高さ: 400px（PC）/ 300px（SP）
- Google Maps埋め込み、またはスタティック画像

**住所:**
- サイズ: 15px、游ゴシック、#333333
- 地図から32px下
- 中央揃え

**最寄駅:**
- サイズ: 14px、游ゴシック、#666666
- 住所から16px下
- 中央揃え

### デザイン詳細

**背景:** #FFFFFF
**余白:** 上下各120px

### Google Maps埋め込みの注意

- グレースケールのカスタムスタイルを適用（可能であれば）
- ズームレベルは15-16程度
- UIは最小限に

## ページ全体のフロー設計

### 視線の流れ

```
1. タイトル: ページの意図を認識
   ↓
2. 会社情報: 基本情報を確認
   ↓
3. 沿革: 歴史を把握（ある場合）
   ↓
4. アクセス: 所在地を確認（ある場合）
```

## バリエーション

### コンパクト版

沿革・アクセスを省略し、会社情報のみのシンプルな構成。
スタートアップや小規模企業向け。

### 統合版

チームページを設けず、会社概要にメンバー紹介を統合。
代表者1名の場合に有効。

## 実装時の注意点

### データ構造

会社情報はsite.jsonで管理:

```json
{
  "company": {
    "name": "株式会社〇〇",
    "representative": "山田 太郎",
    "established": "2003年4月",
    "address": {
      "postalCode": "000-0000",
      "prefecture": "東京都",
      "city": "渋谷区",
      "street": "〇〇1-2-3",
      "building": "〇〇ビル 5F"
    },
    "phone": "03-0000-0000",
    "business": [
      "ブランディング",
      "ウェブサイト制作",
      "コンサルティング"
    ],
    "history": [
      { "year": 2003, "event": "創業" },
      { "year": 2005, "event": "株式会社〇〇 設立" }
    ]
  }
}
```

### 構造化データ

LocalBusinessスキーマを実装:

```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "株式会社〇〇",
  "address": { ... }
}
```
