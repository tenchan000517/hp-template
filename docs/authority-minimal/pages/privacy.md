# プライバシーポリシーページ 詳細設計

**URL**: /privacy

---

## このページの目的

### 専門家①（マーケティング）の視点

**マーケティング目標:**
- 法的要件を満たす（個人情報保護法への準拠）
- 信頼性の証明（プライバシーへの配慮）
- 問い合わせフォームからのリンク先として機能

**コンバージョン設計:**
- 直接的なCVは期待しない
- 「ちゃんとした会社」という印象の補強

### 専門家②（心理学）の視点

**心理的ゴール:**
- **安心感の提供**: 「個人情報は適切に扱われる」
- **透明性**: 何に使うかを明示
- **信頼の補強**: 法令遵守の姿勢

### 専門家③（デザイン）の視点

**デザイン上の役割:**
- 法的文書だが、読みやすさを確保
- Authority Minimalのトーンを維持しつつ、実用的に
- 長文だが、構造化して読みやすく

---

## ユーザーの状態

**このページに来る前の状態:**
- 問い合わせフォームで「同意」リンクをクリック
- または、フッターからアクセス
- プライバシーへの関心がある

**このページで感じてほしいこと:**
- 「きちんとした会社だ」
- 「情報の扱いが明確」
- 「信頼できる」

**このページから離れる時の状態:**
- 問い合わせページへ戻る（フォーム送信のため）
- 離脱（情報確認が完了）

## セクション3: ポリシー本文

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
法的に必要な項目を網羅しつつ、読みやすく構造化。条項ごとに明確な見出しをつけ、必要な情報を探しやすく。

**専門家②（心理学）:**
長文は読まれにくいが、見出しで構造化することで「必要な部分だけ読む」ことを可能に。番号をつけることで順序と網羅性を示す。

**専門家③（デザイン）:**
条項ごとに区切りを入れ、視認性を確保。見出しは目立たせるが、派手にはしない。

### レイアウト

```
+--------------------------------------------------+
|                                                  |
|   1. 個人情報の定義                               |
|                                                  |
|   本方針において「個人情報」とは、個人情報保護法に |
|   定める個人情報を指し、生存する個人に関する情報   |
|   であって...                                     |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   2. 個人情報の収集                               |
|                                                  |
|   当社は、以下の場合に個人情報を収集することが     |
|   あります。                                      |
|                                                  |
|   ・お問い合わせフォームからのご連絡時             |
|   ・サービスのお申し込み時                        |
|   ・採用へのご応募時                              |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   3. 個人情報の利用目的                           |
|                                                  |
|   ...                                            |
|                                                  |
+--------------------------------------------------+
```

### コンテンツ要素

**条項見出し（H2）:**
- サイズ: 20px（PC）/ 18px（SP）、游明朝、#1A1A1A
- 形式: 「1. 〇〇」（番号 + ピリオド + スペース + タイトル）

**条項本文:**
- サイズ: 15px、游ゴシック、#333333
- 行間: 1.9
- 段落間: 16px
- 見出しから24px下

**箇条書き:**
- 形式: 「・」（中黒）
- インデント: 24px
- 項目間: 8px

**区切り線:**
- 幅: 100%、高さ: 1px、色: #E0E0E0
- 条項間に配置

### 必須条項（推奨構成）

1. **個人情報の定義**
2. **個人情報の収集**
3. **個人情報の利用目的**
4. **個人情報の第三者提供**
5. **個人情報の管理**
6. **個人情報の開示・訂正・削除**
7. **Cookieについて**
8. **アクセス解析ツールについて**
9. **本方針の変更**
10. **お問い合わせ先**

### デザイン詳細

**背景:** #FFFFFF

**余白:**
- 左右マージン: 15%
- 条項間: 48px（区切り線を含む）
- 最大幅: 800px、中央揃え

**特徴的な表現:**
- 法的文書だが、読みやすいタイポグラフィ
- 過度な装飾は排除
- 条項番号で構造を明確に

**サイズ変更:**
- 条項見出し: 18px
- 条項本文: 14px

## ポリシー本文のテンプレート

以下は、site.jsonに格納する推奨テンプレート:

```markdown
## 1. 個人情報の定義

本方針において「個人情報」とは、個人情報の保護に関する法律（以下「個人情報保護法」といいます）に定める個人情報を指し、生存する個人に関する情報であって、当該情報に含まれる氏名、生年月日その他の記述等により特定の個人を識別できるもの、または個人識別符号が含まれるものをいいます。

## 2. 個人情報の収集

当社は、以下の場合に個人情報を収集することがあります。

・お問い合わせフォームからのご連絡時
・サービスのお申し込み時
・採用へのご応募時
・その他、当社サービスのご利用時

## 3. 個人情報の利用目的

当社は、収集した個人情報を以下の目的で利用します。

・お問い合わせへの回答
・サービスの提供・改善
・採用選考
・当社からのお知らせ・ご案内
・その他、上記に付随する業務

## 4. 個人情報の第三者提供

当社は、以下の場合を除き、お客様の同意なく個人情報を第三者に提供することはありません。

・法令に基づく場合
・人の生命、身体または財産の保護のために必要な場合
・公衆衛生の向上または児童の健全な育成の推進のために必要な場合
・国の機関等への協力が必要な場合

## 5. 個人情報の管理

当社は、個人情報の正確性を保ち、これを安全に管理します。個人情報の紛失、破壊、改ざん、漏洩等を防止するため、適切なセキュリティ対策を実施します。

## 6. 個人情報の開示・訂正・削除

お客様ご本人から、個人情報の開示・訂正・削除のご請求があった場合、ご本人確認の上、合理的な範囲で速やかに対応します。

## 7. Cookieについて

当社ウェブサイトでは、利便性向上のためにCookieを使用することがあります。Cookieは、お客様のブラウザに保存される小さなテキストファイルです。ブラウザの設定によりCookieを無効にすることも可能ですが、一部機能が制限される場合があります。

## 8. アクセス解析ツールについて

当社ウェブサイトでは、Googleが提供するアクセス解析ツール「Googleアナリティクス」を使用しています。Googleアナリティクスは、Cookieを使用してお客様のウェブサイト利用状況を収集しますが、個人を特定する情報は含まれません。収集されたデータはGoogleのプライバシーポリシーに基づいて管理されます。

## 9. 本方針の変更

当社は、法令の改正その他の理由により、本方針を変更することがあります。変更後の方針は、当ウェブサイトに掲載した時点から効力を生じます。

## 10. お問い合わせ先

個人情報の取扱いに関するお問い合わせは、以下までご連絡ください。
```

## ページ全体のフロー設計

### 視線の流れ

```
1. タイトル: ページの意図を認識
   ↓
2. 前文: 基本姿勢を理解
   ↓
3. 条項: 必要な項目を確認
   ↓
4. 日付: 最新性を確認
   ↓
5. 問い合わせ先: 連絡先を確認
```

## 実装時の注意点

### 法的レビュー

- このテンプレートは参考例であり、実際の使用時には法務担当または弁護士のレビューを推奨
- 業種や取り扱う個人情報によって、追加条項が必要な場合あり

### 更新管理

- 改定日の更新を忘れずに
- 重要な変更時は既存顧客への通知を検討

### SEO

- noindexを設定することが多い
- 検索結果に表示させる必要がないため

### 構造化データ

不要（法的文書に構造化データは通常不要）
