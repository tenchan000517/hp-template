# TOPページ 詳細設計

**URL**: /

---

## このページの目的

### 専門家①（マーケティング）の視点

**マーケティング目標:**
- 3秒以内に「他社とは違う」という第一印象を刻む
- 離脱率は高くてよい（ターゲット外を自然に離脱させる）
- 残った訪問者の「詳しく知りたい」という欲求を喚起する
- 指名検索・紹介で来た人に「期待通りだ」という安心感を与える

**コンバージョン設計:**
- TOPページでの直接コンバージョン（問い合わせ）は期待しない
- 「理念ページ」または「事例ページ」への遷移が成功指標
- CTAは控えめに、ページ最下部に1箇所のみ

### 専門家②（心理学）の視点

**心理的ゴール:**
- **認知的閉合の欲求を刺激**: 情報を出し惜しみすることで「続きを知りたい」
- **第一印象の固定化**: 最初に感じた「品格」がその後の判断基準になる
- **社会的証明の暗示**: 「こういう会社に依頼する人は、良い判断をしている」

**活用する心理効果:**
1. ハロー効果: 美しいデザイン → 仕事も美しいに違いない
2. 希少性の原理: 情報が少ない → 価値が高いに違いない
3. 権威性バイアス: 堂々とした佇まい → 実力があるに違いない

### 専門家③（デザイン）の視点

**デザイン上の役割:**
- このテンプレート全体のトーンを決定する「顔」
- 「静謐な自信」を最も強く表現するページ
- 要素を極限まで削ぎ落とし、余白で語る

**視覚的ゴール:**
- 白と黒のコントラストで、凛とした空気感
- 大胆なタイポグラフィが絵画のように画面を構成
- スクロールするたびに「発見」がある、しかし派手ではない

## ユーザーの状態

**このページに来る前の状態:**
- 紹介または指名検索で来た場合: 「どんな会社だろう」という好奇心
- 一般検索で来た場合: 「この会社は信頼できるか」という疑念
- 競合比較中の場合: 「他社と何が違うのか」という比較意識

**このページで感じてほしいこと:**
- 「この会社は、他とは明らかに違う」
- 「安っぽくない。本物だ」
- 「もう少し知りたい」

**このページから離れる時の状態:**
- 理念ページへ: 「どんな考えを持っている会社か知りたい」
- 事例ページへ: 「実際にどんな仕事をしているか見たい」
- 離脱: 「自分には合わない」（これも成功。ミスマッチを防いでいる）

---

## セクション構成

TOPページは以下の4セクションで構成する:

1. **ファーストビュー（FV）**: キャッチコピーのみ
2. **理念の導入**: 代表の言葉を1文だけ
3. **事業の暗示**: 何をする会社かを3ワードで
4. **お問い合わせ導線**: 控えめなCTA

---

## セクション1: ファーストビュー（FV）

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
FVはサイトの「玄関」。玄関が安っぽければ、その先も安っぽいと判断される。Authority Minimalでは、FVで「価格ではなく価値で勝負する会社だ」と伝える。ここで離脱する人は、そもそもターゲットではない。

**専門家②（心理学）:**
人間は最初の0.5秒で直感的な判断を下す（「薄切り」理論）。この最初の印象が、その後のすべての認知に影響する。高級感を感じさせることで、その後の情報もポジティブに解釈される。

**専門家③（デザイン）:**
FVは「美術館に入った瞬間」に相当する。白い壁、静かな空気、最初に目に入る作品。ここで「この美術館は信頼できる」と感じてもらう。情報を詰め込まないことで、1つ1つの要素に価値が生まれる。

### レイアウト

**構成:**
画面を大きく3つのエリアに分割する非対称レイアウト。

```
+--------------------------------------------------+
|  [ロゴ]                    [ナビゲーション]      | ← ヘッダー（固定）
+--------------------------------------------------+
|                                                  |
|                                                  |
|     本質を、                                      | ← 左1/3の位置にキャッチコピー
|     かたちに。                                    |   2行構成、明朝体72px
|                                                  |
|                                          Essence | ← 右下に英語タグライン
|                                      into Form.  |   欧文セリフ14px、薄いグレー
|                                                  |
|                                                  |
+--------------------------------------------------+
```

**なぜこのレイアウトか:**
- 左寄せにすることで、右側に大きな余白が生まれる
- 右下の英語タグラインが、斜め方向の視線誘導を生む
- 左上→右下の視線の流れは、日本語の読み方向と一致する

**代替案を採用しない理由:**
- 中央配置: よくある配置で差別化できない。また、左右対称は「無難」に見える
- 画像あり: 画像を入れると情報量が増え、「静謐さ」が損なわれる
- 複数行のサブテキスト: 情報過多になり、キャッチコピーの印象が薄まる

### コンテンツ要素

**要素1: メインキャッチコピー**

- 位置: 画面左から15%、垂直方向は中央よりやや上（上から40%の位置）
- サイズ: 72px（PC）/ 36px（SP）
- フォント: 游明朝（または指定の明朝体）
- 字間: 0.15em
- 行間: 1.6
- 色: #1A1A1A

配置理由: 視線は左上から入る。しかし最上部ではなく、少し下げることで「余裕」を演出。上部40%が余白となり、高級感が生まれる。

**要素2: 英語タグライン**

- 位置: 画面右から10%、垂直方向は下から15%
- サイズ: 14px（PC）/ 12px（SP）
- フォント: Cormorant Garamond（欧文セリフ）
- 字間: 0.2em
- 色: #999999（薄いグレー）

配置理由: キャッチコピーと対角線上に配置することで、視線が画面全体を巡る。また、目立たせすぎないことで「添え物」としての役割を保つ。

**要素3: スクロールインジケーター**

- 位置: 画面下中央、下から5%
- 形状: 細い垂直線（高さ40px、幅1px）
- 色: #CCCCCC
- アニメーション: なし（静止したまま）

配置理由: スクロールを促すが、押し付けがましくない。「見たければ見てください」という姿勢。

### デザイン詳細

**背景:**
- 色: #FFFFFF（純白）
- 理由: グレーやベージュでは「やわらかさ」が出てしまう。純白にすることで、緊張感と清潔感を維持

**余白:**
- 上余白: なし（ヘッダーから直結）
- 下余白: 200px（次セクションとの間に大きな呼吸）
- 左余白（コンテンツまで）: 15%（120px程度）
- 右余白: 自動（大きな余白が生まれる）

**なぜこの数値か:**
上余白がないのは、ヘッダーとFVを一体として扱うため。下余白200pxは、「FVが終わった」ことを明確に示すため。通常のセクション間余白（160px）より大きくすることで、FVの特別感を演出。

**特徴的な表現:**
- 背景画像なし（これが特徴）
- オーバーレイなし
- グラデーションなし
- 純粋にタイポグラフィと余白のみで構成

### アニメーション

**使用する/しない:** 使用しない

**理由:**
Authority Minimalでは、FVに動きを入れない。理由は以下:
1. 動きは「注目を集めたい」意図を感じさせる
2. 静止していることで「堂々としている」印象
3. 読み込み直後に最も重要な要素が完成形で表示される信頼感

ただし、ページ遷移時の白フェード（0.3秒）は許容する。

### SP（スマートフォン）での変化

**レイアウト変更:**
- キャッチコピーは中央寄せ（左寄せだと短い行が不自然になるため）
- 英語タグラインはキャッチコピーの直下に配置（対角線配置は解除）

**サイズ変更:**
- キャッチコピー: 36px
- 英語タグライン: 12px
- 下余白: 120px

**省略要素:**
- スクロールインジケーターは非表示（SPでは自然にスクロールするため）

### このセクションでの心理効果

**ユーザーが感じること:**
- 「情報が少ない...でも不思議と不安ではない」
- 「この会社は、自信があるのだな」
- 「安い会社ではなさそうだ」

**次のセクションへの期待:**
- 「どんな会社なんだろう」という好奇心
- キャッチコピーの意味をもっと知りたい気持ち

## セクション2: 理念の導入

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
FVで「他と違う」印象を与えた後、すぐに「何をする会社か」を説明するのは陳腐。まず「どんな考えの会社か」を伝えることで、サービス内容を見る前に共感を生む。価格ではなく価値観で選ばれる企業になるための布石。

**専門家②（心理学）:**
人間は「物語」に引き込まれる生き物。単なる事実の羅列より、代表者の言葉（一人称の語り）のほうが共感を生みやすい。また、「続きは理念ページで」とすることで、認知的閉合の欲求を刺激し、遷移を促す。

**専門家③（デザイン）:**
FVの「静」の後に、長文は入れない。1文だけ、印象的な言葉を配置する。まるで美術館の最初の展示室に、作品解説ではなくアーティストの言葉が掲げられているように。

### レイアウト

**構成:**
画面中央に1文のみ。周囲は全て余白。

```
+--------------------------------------------------+
|                                                  |
|                                                  |
|                                                  |
|         私たちは、答えではなく                     |
|         問いを持つクライアントと共に歩みます。     |
|                                                  |
|                                      [続きを読む →]|
|                                                  |
|                                                  |
+--------------------------------------------------+
```

**なぜこのレイアウトか:**
- 中央配置にすることで、視線が自然に集まる
- 1文しかないことで、その言葉の重みが増す
- 右下の「続きを読む」が、理念ページへの導線

**代替案を採用しない理由:**
- 複数段落: 情報過多。読まずに飛ばされる
- 左寄せ: FVと同じ配置だと単調になる
- 代表者写真を添える: まだ早い。顔は理念ページで初めて見せる

### コンテンツ要素

**要素1: 代表者の言葉（抜粋）**

- 位置: 画面中央、垂直方向も中央
- サイズ: 24px（PC）/ 18px（SP）
- フォント: 游明朝
- 字間: 0.05em
- 行間: 2.0
- 色: #333333
- 最大幅: 600px（これ以上広がらない）

配置理由: 本文サイズ（16px）より大きいが、見出しサイズ（32px）よりは小さい。引用文として特別な扱いをする。

**要素2: 続きを読むリンク**

- 位置: 引用文の右下、48px下
- サイズ: 14px
- フォント: 游ゴシック
- 色: #666666
- ホバー時: テキストカラーが#333333に、下線が出現

配置理由: 控えめなサイズで、押し付けがましくない。「読みたければどうぞ」という姿勢。

### デザイン詳細

**背景:**
- 色: #FFFFFF（純白、FVと同じ）
- 理由: 背景を変えないことで、FVからの連続性を保つ

**余白:**
- 上余白: 200px（FVからの大きな余白）
- 下余白: 160px（通常のセクション間余白）
- 左右余白: 自動（テキストは600px幅で中央揃え）

**特徴的な表現:**
- 引用符（「」）は使わない。書体と配置で引用であることを示す
- 署名も入れない（代表者の言葉であることは、理念ページで明かす）

### アニメーション

**使用する/しない:** テキストのみフェードイン

**具体的な動き:**
- トリガー: 要素がビューポートに入った時（-100pxのマージン）
- 動き: opacity 0→1、y 20px→0
- 時間: 0.6秒
- イージング: cubic-bezier(0.4, 0, 0.2, 1)
- 1回のみ再生

**意図:**
FVは静止しているが、スクロール後の最初の要素にのみフェードインを使う。「スクロールしたことで、新しい情報が現れた」という発見の演出。ただし、控えめな動きに留める。

### SP（スマートフォン）での変化

**レイアウト変更:**
- 変更なし（中央配置のまま）
- 左右のパディングを24pxに

**サイズ変更:**
- 引用文: 18px
- 続きを読む: 13px
- 上余白: 120px
- 下余白: 100px

### このセクションでの心理効果

**ユーザーが感じること:**
- 「言葉に重みがある」
- 「この会社は、表面的ではない」
- 「代表者に興味が湧いた」

**次のセクションへの期待:**
- 理念ページへの遷移欲求（もっと知りたい）
- または「何をする会社か」を知りたい気持ち

## セクション3: 事業の暗示

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
理念だけでは「何屋さん」かわからない。ここで事業内容を示すが、詳細は説明しない。3つのキーワードで「ざっくり何をする会社か」を伝え、詳細はサービスページに委ねる。これにより、サービスページへの遷移を促す。

**専門家②（心理学）:**
人間は「3」という数字に安心感を覚える（マジックナンバー3）。3つのキーワードは覚えやすく、選択の負担も感じさせない。また、詳細を省くことで「もっと知りたい」欲求を刺激。

**専門家③（デザイン）:**
3つの言葉を等間隔に並べるのではなく、非対称に配置する。ここでもグリッドから外れることで、「人間が設計した」印象を与える。

### レイアウト

**構成:**
3つのサービスカテゴリを、斜めのラインを意識して配置。

```
+--------------------------------------------------+
|                                                  |
|   Strategy                                       |
|   戦略設計                                        |
|                                                  |
|                          Design                  |
|                          デザイン                 |
|                                                  |
|                                      Branding    |
|                                      ブランディング|
|                                                  |
|                                                  |
|         [サービス詳細 →]                          |
|                                                  |
+--------------------------------------------------+
```

**なぜこのレイアウトか:**
- 斜め配置により、視線が左上→右下へ自然に流れる
- 均等配置ではないため、「意図がある」配置に見える
- 各カテゴリ間に大きな余白があり、高級感

**代替案を採用しない理由:**
- 3カラム均等配置: AIデザインの典型。無個性
- カード型: 情報過多。ここでは詳細を見せない
- アイコン付き: 装飾的すぎる。Minimalの原則に反する

### コンテンツ要素

**要素1-3: サービスカテゴリ（3つ）**

各カテゴリの構成:
- 英語表記: 14px、Cormorant Garamond、#999999
- 日本語表記: 20px、游ゴシック Medium、#333333
- 英語→日本語の間隔: 8px

配置:
- カテゴリ1: 左から10%、上から20%
- カテゴリ2: 左から45%、上から45%
- カテゴリ3: 左から70%、上から70%

配置理由: 対角線上に配置することで、視線誘導と動きのあるレイアウトを実現。

**要素4: サービス詳細リンク**

- 位置: 最後のカテゴリから80px下、中央揃え
- サイズ: 14px
- フォント: 游ゴシック
- 色: #666666
- ホバー時: 下線出現

### デザイン詳細

**背景:**
- 色: #F5F5F5（極薄いグレー）
- 理由: FV〜理念までの白から変化をつけ、セクションの区切りを明示

**余白:**
- 上余白: 160px
- 下余白: 160px
- セクション高さ: 最低600px（要素が少ないため、高さで空間を確保）

**特徴的な表現:**
- 境界線やカードなし
- 要素同士は余白のみで分離
- 背景色の変化のみで「別のセクション」と認識させる

### アニメーション

**使用する/しない:** 使用しない

**理由:**
3つの要素がバラバラに動くと煩雑。静止した状態で見せることで、全体の配置バランスを感じてもらう。

### SP（スマートフォン）での変化

**レイアウト変更:**
- 斜め配置は解除し、縦に積む
- すべて左寄せ
- 各カテゴリ間の余白: 48px

**サイズ変更:**
- 英語表記: 12px
- 日本語表記: 18px
- 上下余白: 100px

### このセクションでの心理効果

**ユーザーが感じること:**
- 「ざっくり何をする会社かわかった」
- 「詳細を見たいかも」
- 「3つに絞っているということは、専門性があるのだろう」

**次のセクションへの期待:**
- サービスページへの遷移（詳細を知りたい）
- またはお問い合わせの検討

## セクション4: お問い合わせ導線

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
TOPページの最後にCTAを配置するのはセオリーだが、Authority Minimalでは「控えめなCTA」が原則。ここで強くプッシュするのではなく、「準備ができたらどうぞ」という姿勢。

**専門家②（心理学）:**
「今すぐ問い合わせ」の圧力は、高単価サービスでは逆効果。むしろ「急かされない」ことで、「この会社は自信がある」「自分のペースで検討できる」という安心感を与える。

**専門家③（デザイン）:**
派手なボタンではなく、テキストリンクに近い控えめなCTA。しかし確実に目に入る位置に配置する。「お問い合わせはこちら」ではなく、「プロジェクトについてお聞かせください」のような表現。

### レイアウト

**構成:**
中央に1行のテキストと、その下に控えめなボタン。

```
+--------------------------------------------------+
|                                                  |
|                                                  |
|           プロジェクトについて                     |
|           お聞かせください。                       |
|                                                  |
|              [Contact →]                         |
|                                                  |
|                                                  |
+--------------------------------------------------+
```

**なぜこのレイアウトか:**
- 中央配置で、視線が自然に集まる
- 2行のテキスト + ボタンというシンプルな構成
- 大きなボタンではなく、テキストリンク風のボタン

**代替案を採用しない理由:**
- 大きなCTAボタン: 圧力が強すぎる
- フォーム埋め込み: TOPページでは早い
- 電話番号併記: 権威性を損なう（気軽さを出しすぎる）

### コンテンツ要素

**要素1: お声がけテキスト**

- サイズ: 28px（PC）/ 22px（SP）
- フォント: 游明朝
- 色: #333333
- 行間: 1.6
- 中央揃え

配置理由: 見出しサイズ（32px）より少し小さく、主張しすぎない。

**要素2: Contactボタン**

- 位置: テキストから48px下、中央揃え
- サイズ: 14px
- フォント: Cormorant Garamond（英字）
- 色: #333333
- ボーダー: 1px solid #333333
- パディング: 16px 48px
- ホバー時: 背景#333333、テキスト#FFFFFF

配置理由: 日本語ではなく英語表記にすることで、品位を保つ。ボーダーのみのシンプルなボタン。

### デザイン詳細

**背景:**
- 色: #FFFFFF（白に戻す）

## ページ全体のフロー設計

### 視線の流れ

```
1. ヘッダー: ロゴを認識、ナビゲーションを確認
   ↓
2. FV: キャッチコピーで第一印象を形成
   ↓ スクロール
3. 理念の導入: 代表者の言葉で共感形成
   ↓ スクロール
4. 事業の暗示: 何をする会社か大枠を理解
   ↓ スクロール
5. CTA: 問い合わせの選択肢を認識
   ↓
6. 離脱または遷移
```

### 期待される行動

**最優先の遷移先:**
1. 理念ページ（代表者の考えを深く知りたい人）
2. 事例ページ（実績を確認したい人）

**許容される行動:**
3. サービスページ（具体的なサービス内容を知りたい人）
4. 離脱（ターゲット外）

**この段階で期待しないこと:**
- 問い合わせ（まだ情報が足りない）
- 価格の確認（このテンプレートでは価格ページを設けない）

## 実装時の注意点

### パフォーマンス

- 画像なしのため、ページサイズは極めて軽量に
- Webフォントの読み込みを最適化（font-display: swap）
- FVは即座に表示されるよう、Critical CSSを適用

### アクセシビリティ

- キャッチコピーに適切なaria-labelを設定
- フォーカス可能な要素に明確なフォーカスリングを
- 色のコントラスト比4.5:1以上を確保

### SEO

- title: [会社名] | [キャッチコピー]
- description: 理念の導入テキストを使用
- OGP画像: 白背景 + キャッチコピーのテキストのみ
