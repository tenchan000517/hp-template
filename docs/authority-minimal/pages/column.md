# コラムページ 詳細設計

**URL**: /column（一覧）、/column/[slug]（詳細）

---

## このページの目的

### 専門家①（マーケティング）の視点

**マーケティング目標:**
- 専門性を「売り込み」ではなく「情報提供」で証明する
- SEO流入の受け皿（指名検索以外からの接点）
- 潜在顧客の育成（今すぐ客ではなく、将来の客）
- 他社との差別化（独自の視点・知見の発信）

**コンバージョン設計:**
- コラムページでの直接的なCVは期待しない
- 代わりに「この会社は詳しそうだ」という認識を形成
- 複数記事を読んでもらい、サービスページや事例ページへ遷移

### 専門家②（心理学）の視点

**心理的ゴール:**
- **専門家としての認知**: 「この人は本当に詳しい」
- **返報性の原理**: 無料で価値ある情報を提供 → 依頼で返したくなる
- **認知的容易性**: 複雑な内容をわかりやすく解説 → 信頼感

**活用する心理効果:**
1. 教育効果: 教える立場=権威がある
2. 一貫性の原理: 継続的な発信=信頼できる
3. 好意の獲得: 役立つ情報を無料で提供=好感を持たれる

### 専門家③（デザイン）の視点

**デザイン上の役割:**
- 「読み物」として完結する独立したページ群
- ブログ的なカジュアルさではなく、「論考」としての格
- 余白を活かした読みやすいレイアウト

**視覚的ゴール:**
- 本文に集中できる、ノイズのない画面
- 記事ごとの統一感（テンプレート的な一貫性）
- 装飾を排した、内容で勝負するデザイン

---

## ユーザーの状態

**このページに来る前の状態:**
- 検索エンジンから特定のキーワードで流入
- または、他ページから「もっとこの会社を知りたい」と遷移
- 業界の知識を求めている（課題意識がある）

**このページで感じてほしいこと:**
- 「この記事は役に立つ」
- 「この会社は本当に詳しい」
- 「他の記事も読んでみよう」

**このページから離れる時の状態:**
- 別のコラムへ: 「もっと読みたい」
- サービスページへ: 「どんなサービスをしているか知りたい」
- ブックマーク: 「また来よう」

## ページ構成

コラムページは2つの階層で構成する:

1. **コラム一覧ページ** (/column): 全記事のリスト
2. **コラム詳細ページ** (/column/[slug]): 個別記事

## セクション2: カテゴリフィルター（任意）

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
記事数が多い場合（10件以上）、カテゴリで絞り込めると便利。ただし、カテゴリは3-4個に抑える。

**専門家②（心理学）:**
選択肢が多すぎると選べない。シンプルなフィルタリングは認知負荷を下げる。

**専門家③（デザイン）:**
テキストリンク形式のフィルター。タブやボタンではなく、控えめな表現。

### レイアウト

```
+--------------------------------------------------+
|                                                  |
|   すべて   戦略   デザイン   ブランディング       |
|                                                  |
+--------------------------------------------------+
```

### コンテンツ要素

**フィルターリンク**
- サイズ: 14px、游ゴシック
- 非選択時: #999999
- 選択時: #333333、下線1px
- 間隔: 32px

### デザイン詳細

**背景:** #FFFFFF
**余白:** 上0、下48px

---

## セクション3: 記事一覧

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
最新記事を上に、時系列で並べる。タイトルとリード文で興味を引き、クリックを促す。

**専門家②（心理学）:**
日付を表示することで「最近も更新している」という活動感を示す。更新が止まっていると信頼を損なう。

**専門家③（デザイン）:**
カード型ではなく、リスト形式。余白で区切る。サムネイル画像は使用しない（言葉で勝負）。

### レイアウト

**構成:**
縦に並ぶシンプルなリスト形式。

```
+--------------------------------------------------+
|                                                  |
|   2024.01.15                                     |
|   なぜ「安さ」で勝負すると負けるのか              |
|   価格競争に陥らないためのブランディング戦略...   |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   2024.01.08                                     |
|   リブランディングのタイミングはいつか            |
|   創業から10年、変革期を迎えた企業がすべきこと... |
|                                                  |
|   ─────────────────────────────────────          |
|                                                  |
|   2023.12.22                                     |
|   デザインの「良し悪し」を判断する基準            |
|   主観を超えた評価軸の作り方...                   |
|                                                  |
+--------------------------------------------------+
```

**なぜこのレイアウトか:**
- 画像がないことで「読み物」としての格を出す
- リスト形式で一覧性が高い
- 日付→タイトル→リード文の順で情報を整理

**代替案を採用しない理由:**
- カード型グリッド: ブログ的になる。カジュアルすぎる
- サムネイル画像付き: 画像選定が難しく、質が落ちがち
- 無限スクロール: 一覧性が低い

### コンテンツ要素

**各記事の構成:**

日付:
- サイズ: 13px、Cormorant Garamond、#999999
- 形式: 2024.01.15

タイトル:
- サイズ: 22px（PC）/ 18px（SP）、游明朝、#1A1A1A
- 日付から16px下
- ホバー時: 色が#666666に変化

リード文:
- サイズ: 14px、游ゴシック、#666666
- タイトルから12px下
- 最大2行（超過は...で省略）
- 最大幅: 600px

区切り線:
- 幅: 100%、高さ: 1px、色: #E0E0E0
- 記事から48px下

**記事間余白:** 48px（区切り線を含む）

### デザイン詳細

**背景:** #FFFFFF
**左右マージン:** 15%

**特徴的な表現:**
- 画像なし
- 区切り線のみで分離
- テキストに集中

### アニメーション

**使用する/しない:** 使用しない

### SP（スマートフォン）での変化

**サイズ変更:**
- タイトル: 18px
- リード文: 13px
- 左右パディング: 24px

# コラム詳細ページ

## セクション構成

1. **記事ヘッダー**: タイトルと日付
2. **本文**: 記事内容
3. **著者情報**: 執筆者プロフィール
4. **関連記事**: 他のおすすめ記事
5. **CTA**: サービスページへの導線

---

## セクション1: 記事ヘッダー

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
記事タイトルは最も重要な要素。検索結果やSNSシェア時にも表示される。明確で興味を引くタイトルを大きく表示。

**専門家②（心理学）:**
タイトルで「読む価値がある」と判断させる。日付があることで新鮮さを確認できる。

**専門家③（デザイン）:**
余白を大きく取り、タイトルに重みを与える。FVのように、上部50%を余白とする。

### レイアウト

**構成:**

```
+--------------------------------------------------+
|                                                  |
|                                                  |
|                                                  |
|   2024.01.15                                     |
|   Column                                         |
|                                                  |
|   なぜ「安さ」で勝負すると                        |
|   負けるのか                                      |
|                                                  |
|   価格競争から脱却するためのブランディング戦略    |
|                                                  |
+--------------------------------------------------+
```

### コンテンツ要素

**要素1: 日付**
- サイズ: 13px、Cormorant Garamond、#999999
- 位置: 左から15%、上から45%

**要素2: カテゴリ（Column）**
- サイズ: 14px、Cormorant Garamond、#999999
- 日付から8px下

**要素3: タイトル**
- サイズ: 40px（PC）/ 28px（SP）、游明朝、#1A1A1A
- カテゴリから32px下
- 行間: 1.5
- 最大幅: 700px

**要素4: サブタイトル（任意）**
- サイズ: 16px、游ゴシック、#666666
- タイトルから24px下

### デザイン詳細

**背景:** #FFFFFF
**余白:** 上 ビューポート45%（最低280px）、下 80px

## セクション2: 本文

### なぜこのセクションが必要か

**専門家①（マーケティング）:**
記事の本体。専門性を発揮しつつ、読みやすさを確保。長すぎる記事は読まれないが、薄い内容は価値がない。2000-4000文字程度が目安。

**専門家②（心理学）:**
読みやすさは信頼感に直結。適切な改行、見出しによる構造化、余白による視覚的休息。

**専門家③（デザイン）:**
本文に最適化されたタイポグラフィ。1行の文字数を制限し、行間を広く。見出しと本文の強弱を明確に。

### レイアウト

**構成:**

```
+--------------------------------------------------+
|                                                  |
|   多くの企業が「価格で勝負」しようとします。      |
|   しかし、それは最も危険な戦略です。              |
|                                                  |
|   価格で選ばれるということは、より安い競合が      |
|   現れた瞬間に顧客を失うことを意味します。        |
|                                                  |
|                                                  |
|   ── 価格競争の本当の問題                        |
|                                                  |
|   価格競争の問題は、利益率の低下だけでは          |
|   ありません。より深刻なのは...                   |
|                                                  |
|                                                  |
|   ── ブランドで選ばれる企業になるには            |
|                                                  |
|   では、価格ではなくブランドで選ばれるためには    |
|   何が必要でしょうか...                           |
|                                                  |
+--------------------------------------------------+
```

### コンテンツ要素

**要素1: 本文段落**

- サイズ: 16px（PC）/ 15px（SP）
- フォント: 游ゴシック
- 色: #333333
- 行間: 2.1
- 字間: 0.02em
- 段落間余白: 32px
- 最大幅: 680px
- 中央揃え配置（左右均等マージン）

**要素2: 中見出し（H2）**

- サイズ: 22px（PC）/ 18px（SP）
- フォント: 游明朝
- 色: #1A1A1A
- 装飾: 先頭に「── 」（ダッシュ2つ + 半角スペース）
- 上余白: 64px
- 下余白: 24px

**要素3: 小見出し（H3）**

- サイズ: 18px（PC）/ 16px（SP）
- フォント: 游ゴシック Medium
- 色: #333333
- 上余白: 48px
- 下余白: 16px

**要素4: 引用**

- サイズ: 15px
- フォント: 游明朝
- 色: #666666
- 左ボーダー: 2px solid #E0E0E0
- 左パディング: 24px
- 行間: 2.0

**要素5: 画像（任意）**

- 最大幅: 100%（本文幅に合わせる）
- 上下余白: 48px
- キャプション: 13px、游ゴシック、#999999

### デザイン詳細

**背景:** #FFFFFF
**左右マージン:** 計算上、本文幅680pxが中央に来るように

**特徴的な表現:**
- 見出しの「── 」装飾
- 引用の控えめなスタイル
- 画像は必要最小限

### アニメーション

**使用する/しない:** 使用しない

### SP（スマートフォン）での変化

**サイズ変更:**
- 本文: 15px
- 中見出し: 18px
- 左右パディング: 24px

## 実装時の注意点

### 記事データ構造

```json
{
  "slug": "why-price-competition-fails",
  "title": "なぜ「安さ」で勝負すると負けるのか",
  "subtitle": "価格競争から脱却するためのブランディング戦略",
  "date": "2024-01-15",
  "category": "branding",
  "author": "yamada",
  "content": "Markdown形式の本文"
}
```

### SEO対策

- title: [記事タイトル] | コラム | [会社名]
- description: サブタイトルまたは本文冒頭120文字
- OGP画像: 白背景 + タイトルテキスト

### 更新頻度

- 月1-2本程度の更新を推奨
- 質 > 量
- 長期間更新がない場合、コラムセクション自体を非表示にする選択肢も
