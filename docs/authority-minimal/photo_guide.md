# Authority Minimal 撮影ディレクション

このドキュメントでは、Authority Minimalテンプレートのサイトを完成させるために必要な**全ての素材**を網羅的に記載する。

---

## 5-1. 撮影トーン＆マナー

### コンセプト

**「静謐な自信」**

このサイトの写真は、派手さや華やかさではなく、「静けさ」と「品格」を表現する。被写体が持つ本質的な魅力を、余計な演出なしに引き出す。

### 基本トーン

| 要素 | 指定 |
|------|------|
| 全体の印象 | 落ち着いた、上質な、知的な |
| 色調 | 彩度控えめ、ナチュラル、やや冷たい寄り |
| コントラスト | 中程度（強すぎず弱すぎず） |
| 明るさ | やや明るめ（白背景の写真は特に） |
| 雰囲気 | 清潔感、プロフェッショナル、信頼感 |

### 避けるべきこと

- 過度なレタッチ（肌の加工、色味の極端な変更）
- ドラマチックな陰影（人物写真）
- 暖色系のカラーグレーディング
- スナップ的なカジュアルさ
- 作り込みすぎた「広告写真」感
- 背景の乱雑さ

### カラーパレット

写真全体を通じて、以下のカラーパレットを意識:

- 白（背景、シャツ）
- グレー（背景、スーツ）
- ネイビー（服装のアクセント）
- ベージュ・ブラウン（木材、オフィス家具）
- グリーン（観葉植物、控えめに）

**避ける色:**
- 原色（赤、黄、ビビッドな青）
- パステルカラー
- 蛍光色

## 5-3. 撮影アングル・構図指示

### 人物ポートレート（代表者・メンバー共通）

```
構図:
┌─────────────────┐
│                 │
│    ┌─────┐      │
│    │     │      │  ← 頭上に余白（10%程度）
│    │ 顔  │      │
│    │     │      │
│    └──┬──┘      │
│       │          │
│    ┌──┴──┐      │  ← バストアップ
│    │ 胸  │      │    （胸から上）
│    └─────┘      │
│                 │
└─────────────────┘

アングル: 正面やや斜め（15-30度）
目線: カメラ目線
背景: 白または薄いグレー（グラデーションなし）
```

**ライティング:**
- メインライト: 顔の斜め上45度から
- フィルライト: 反対側から、メインの半分の光量
- 背景は均一に照らす
- 顔に強い影を作らない

**表情:**
- 代表者（メイン）: 真剣さと自信を感じる表情。口角は上げすぎない
- 代表者（サブ）: 軽い微笑み。親しみやすさを感じる程度
- メンバー: 自然な微笑み。リラックスした雰囲気

### オフィス・空間写真

```
構図（広角）:
┌─────────────────────────────┐
│                             │
│    ┌───────────────┐       │
│    │               │       │
│    │   メイン被写体  │       │  ← 部屋の中心
│    │   （テーブル等）│       │
│    │               │       │
│    └───────────────┘       │
│                             │
│  ← 左右に余白を残す →        │
└─────────────────────────────┘

アングル: 目線の高さ、または少し高めから
```

**ポイント:**
- 無人で撮影（人物は別途合成しない）
- 自然光を活かす（窓からの光）
- デスクは整理整頓された状態
- 観葉植物などの「生」の要素を1つ入れる
- モニターの電源はオフ

### 成果物写真

**ロゴ・ブランドアイテムの場合:**

```
構図:
┌─────────────────────────────┐
│                             │
│                             │
│         ┌─────┐             │
│         │ロゴ │             │  ← 中央配置
│         └─────┘             │
│                             │
│                             │
│    名刺・封筒などの          │  ← 下部に補助アイテム
│    展開物（斜め配置）         │
│                             │
└─────────────────────────────┘

背景: 白またはグレー
ライティング: 均一、影を最小限に
```

**ウェブサイトの場合:**

```
構図（モックアップ）:
┌─────────────────────────────┐
│                             │
│      ┌─────────────┐        │
│      │             │        │
│      │   PC画面    │        │  ← MacBookなどの
│      │             │        │    モックアップに
│      └─────────────┘        │    配置
│                             │
│            ┌───┐            │
│            │SP │            │  ← スマホも添える
│            └───┘            │
│                             │
└─────────────────────────────┘

背景: 白または薄いグレー
角度: やや俯瞰（15-30度）
```

## 5-4. 人物別撮影指示

### 代表者

**撮影枚数:** 最低20カット（セレクト用）
**採用枚数:** 2枚（メイン1枚 + サブ1枚）

**服装:**
- スーツまたはジャケット
- 白または淡いブルーのシャツ
- ネクタイはなくてもよい
- 黒・ネイビー・グレー系

**NG:**
- 派手な柄物
- カジュアルすぎる服装（Tシャツ等）
- アクセサリーの目立ちすぎ

**ヘアメイク:**
- 自然な仕上がり
- テカリを抑える程度のパウダー
- 髪は整える程度

### メンバー

**撮影枚数:** 各10カット（セレクト用）
**採用枚数:** 各1枚

**服装:**
- 代表者との統一感を意識
- 白シャツを基本に
- 業種に応じたカジュアルさは許容

**全員共通の統一ルール:**
- 背景色: 全員同じ
- カメラとの距離: 全員同じ
- ライティング: 全員同じ

## 5-5. 成果物撮影指示

### ロゴ・ブランドアイテム

**撮影対象:**
- ロゴマーク（単体）
- 名刺
- 封筒・レターヘッド
- パンフレット
- サイン・看板
- パッケージ（あれば）

**撮影セット:**
- 白またはグレーの背景紙
- 天板（木目またはコンクリート風）
- 自然光または定常光

**配置パターン:**

1. **単体配置**: ロゴだけを中央に
2. **グループ配置**: 名刺・封筒・レターヘッドを斜めに重ねる
3. **使用シーン**: 名刺を手に持つ等（手のみ写す）

### ウェブサイト

**撮影ではなくモックアップで対応**

使用ツール:
- Figmaのモックアッププラグイン
- Mockup Scenesなどのストック素材
- 実機での撮影（MacBook + iPhone）

**推奨モックアップ:**
- MacBook Pro（シルバー）: 斜め45度から
- iPhone（黒）: MacBookの隣に配置
- 背景: 白または木目

### 製品・プロダクト

**撮影環境:**
- 白ホリゾント（白背景スタジオ）
- または自然光が入る室内

**アングル:**
1. 正面（メインカット）
2. 斜め45度
3. ディテール（ロゴ部分等）
4. 使用シーン（可能であれば）

## 5-7. 納品形式

### ファイル形式

| 用途 | 形式 | 解像度 | カラー |
|------|------|--------|--------|
| 元データ | RAW + JPEG | 撮影解像度そのまま | Adobe RGB |
| Web用（大） | JPEG / WebP | 2400px（長辺） | sRGB |
| Web用（中） | JPEG / WebP | 1200px（長辺） | sRGB |
| サムネイル | JPEG / WebP | 600px（長辺） | sRGB |
| OGP用 | JPEG | 1200×630px | sRGB |

### ファイル命名規則

```
[種類]_[用途]_[番号].[拡張子]

例:
portrait_yamada_01.jpg      // 代表者ポートレート
portrait_suzuki_01.jpg      // メンバーポートレート
office_meeting_01.jpg       // オフィス（ミーティングルーム）
case_abc_main_01.jpg        // 事例Aメインビジュアル
case_abc_sub_01.jpg         // 事例Aサブビジュアル
ogp_default.jpg             // OGP画像
```
